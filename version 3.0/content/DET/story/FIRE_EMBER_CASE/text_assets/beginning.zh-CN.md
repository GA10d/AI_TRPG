# 游戏规则简介

你将以一名深陷部落纷争的调查者身份，踏入《罪案边界》所构筑的严酷世界。这是一套完全遵循现实逻辑与刑侦原则的规则体系，你的每一个行动——无论是勘查现场、盘问族人、追踪痕迹，还是梳理时间与动机——都将基于已获得的证据链进行裁定。

这个世界没有神明视角的提示，没有凭空闪现的关键线索。观察力的差异会让你在同样的兽皮帘前看见不同的破绽，交涉技巧的高低会在篝火边的质问中决定对方心理防线的崩溃程度，逻辑推理的强弱则直接关系到你能否从众人矛盾的说辞中拼出唯一的时间线。而一旦误判，证据之间的冲突、证言的微妙破绽、时间的无情流逝，都会让你一步步陷入更深的迷雾。

主持者只会客观呈现你所发现的一切，以程序化的反馈与高度写实的裁定推动案件。答案不会被告知，真相只能在证据闭环与人物动机的彻底浮现后，由你亲手撕开。

# 背景故事概述

冬日渐近，黑石部落的储藏洞里，守夜人长耳发现了岩骨的尸体。

这不是普通的死亡。

洞口的兽皮帘被人用石刀整齐割开——切口平直，绝非野兽所为。散落一地的野栗之间，一道拖拽痕迹蜿蜒着延伸向洞穴深处的暗影。火把在岩壁上剐蹭出的烟灰痕迹还很新鲜，而一把熟悉的石斧——部落共有之物——被丢弃在不起眼的角落。

岩骨，这个年轻气盛、最有希望成为下一任狩猎队长的男人，这个在火塘边与老祭祀谈过话后就再没人公开见过他的人，死在月升至树梢的时刻。

而那一夜，部落东侧确实响起过狼嚎。

但奇怪的是，守夜人长耳坚称，当晚狼群根本没有靠近部落。

你有理由怀疑，这不是意外。

部落里每个人都有自己的说法，而那些说法背后，是不能轻易说清的事：

狩猎队长的人选悬而未决，岩骨与石牙曾为此当众争执。石牙的屋顶茅草有蹬踏痕迹，但他声称自己那晚在屋内修理弓弦。老祭祀的女儿鹿眼，即将被许配给岩骨——有人看见她在那之后独自出入禁地后山，采集一种只有她知道的蛇薄荷。烧陶的黑手，双手总是沾着洗不掉的红黏土，他的陶窑深处藏着什么，谁都不让靠近。

而你，作为被卷入这一切的见证者，必须在这盘根错节的利益、恐惧与谎言中，找到岩骨之死的真相。

火塘的余烬尚温。死亡的阴影正随着冬日渐近而蔓延，部落内部的裂痕也在暗处生长。每一步犹豫，都可能让局势变得更加危险。

# 游戏目的

你的核心任务是通过缜密的调查，拼合岩骨死亡当夜的全部真相。

你需要走访储藏洞、火塘、黑手的陶坊、禁地后山、石牙的住所、老祭祀的棚屋等关键地点。你需要询问每一个知情人，对比他们的证词，寻找物证之间的矛盾，重建案发当晚的真实时间线。你要从狩猎队长之争的权力角力、从鹿眼对婚姻的抗拒、从黑手对陶窑的守护、从老祭祀的沉默中，剥离出那块决定性的拼图。

在风险持续升级之前，在所有人咬死自己的说法之前，你必须做出正确的判断。指认错误不仅无法抓住真凶，还会让真正的杀机永远沉入火塘的灰烬之下。

# 角色确认

你将扮演部落中的调查者。以下是可供选择或分配的角色定位，每位角色在观察、交涉、逻辑与追踪方面有不同侧重，请确认你的角色，或与同伴协商分工：

**长耳**（守夜人/观察优势型）——你是第一个发现尸体的人。你的眼睛习惯了黑夜，细节在你眼前无所遁形，但你的盘问技巧或许不如感官那般敏锐。

**石牙**的友人（交涉优势型）——你在部落中周旋于不同小团体之间，对人的情绪波动与话语漏洞有天然的直觉。你能打开别人紧闭的嘴，但在细微的物证面前，你容易忽略那些沉默的东西。

**老祭祀的助手**（逻辑优势型）——你熟悉部落的规矩与历史的脉络，擅长将破碎的信息拼接成完整的叙事。证词时间线的矛盾在你眼中就像黑夜里的篝火一样醒目，但在与充满敌意的族人直接对峙时，你的力量并不占优。

**追踪者**（追踪优势型）——你能分辨出泥土上是草鞋印还是赤足印，能看出拖拽痕迹是挣扎还是搬运。你对野外痕迹的阅读能力无人能及，但进入室内与人交涉时，你常常觉得有力无处使。

请确认你所担任的角色，以及你率先想要调查的方向。火塘的余烬尚在，但留给你的时间，正在加速流逝。
