# 游戏规则简介

这是一场**纯现实主义刑侦调查**。

你将以卷入案件核心的调查者身份行动，在有限信息、有限时间与不断上升的风险中，依靠**观察、询问、推理、比对证词、重建时间线**来推进案件。这里没有超自然力量，没有命运偏袒，也不会有人替你补全关键空白。

你需要牢记以下原则：

- **一切判断都必须建立在已获得的信息之上。**
- **线索不会自动指向答案**，它们只会彼此印证、冲突，或暴露新的疑点。
- **错误不会被直接宣告**，而会表现为证据链断裂、证言矛盾、人物警觉、时间流失与调查受阻。
- **NPC 可能合作，也可能隐瞒、误导或利用你。**
- 主持将以**客观、程序化、写实**的方式反馈结果，不剧透、不暗示正确答案，只依据你的行动与现有信息推进局势。

请谨慎提问，谨慎判断。  
在这场调查里，**你知道什么，比你觉得什么更重要。**

# 背景故事概述

**1887年，伦敦。**

深秋的浓雾像一层潮湿而冰冷的裹尸布，缓慢吞没街巷、马车、煤气灯，也吞没了许多本不该见光的秘密。就在这样的夜里，汉普斯特德区的**布莱克伍德庄园**爆发了一桩命案。

庄园主人死在自己的书房中。  
他趴伏在红木书桌上，背后插着一把属于家族旧藏的**银柄拆信刀**。房门从内部反锁，钥匙则留在死者口袋中——这让现场看上去像一间近乎无懈可击的密室。

但真正令人不安的，不是那把刀，而是那些**不合常理的细节**：

- 死者右手中攥着半张被撕碎的纸片，上面残留着“**15号仓库**”“**货**”的字样；
- 他的指甲缝里，残留着**深绿色、质地粗糙的纤维**，与书房内常见材质并不吻合；
- 庄园中的人各自给出了看似合理的说法，却又都在某些细节上显得过于平静、过于完整，或过于刻意。

其中，死者的妻子**伊莎贝拉·布莱克伍德**尤其令人难以忽视。她神情憔悴，自称昨夜九点便因头痛回房休息，没有听见异常动静；可她的卧室正对书房，且她说话时指尖微颤，左手无名指上还留有一圈像是刚摘下戒指的淡白痕迹。

这起案件绝不只是“一刀致命”那么简单。  
在这座被雾气与礼仪包裹的庄园里，**死亡只是开始**。

# 游戏目的

你的任务是围绕这起书房命案展开调查，并逐步回答以下问题：

- 这间反锁的书房里，究竟发生了什么？
- 死者临死前留下的纸片，指向了谁，或什么交易？
- 那些不属于书房的绿色纤维，从何而来？
- 庄园中的谁在隐瞒事实，谁在说谎，谁又只是害怕被牵连？
- 这起谋杀背后，真正推动事件发展的动机是什么？

你需要通过：

- 勘查现场；
- 询问相关人员；
- 整理时间线；
- 比对证词与物证；
- 识别矛盾、压力与利益关系；

来逐步拼合出一条**能够自洽、可被证据支撑的真相路径**。

但请注意：  
随着调查深入，局势会持续升级。你的试探可能惊动隐藏者，你的误判可能让关键线索失去时机，而某些人也未必愿意看到真相被完整揭开。

# 角色确认

你将扮演：

**一名被卷入布莱克伍德庄园命案核心的见证者与推动者。**

你不是高高在上的全知侦探，也不是置身事外的旁观者。你身处现场，必须在信息残缺、气氛压抑、人物各怀心思的环境中作出判断。你具备基本的调查能力，包括但不限于：

- 对现场细节的观察；
- 对证词漏洞的辨识；
- 对人物情绪与动机的试探；
- 对时间、距离、行为逻辑的重建；
- 在压力下继续推进调查的决断力。

你的每一步都可能改变案件走向。
