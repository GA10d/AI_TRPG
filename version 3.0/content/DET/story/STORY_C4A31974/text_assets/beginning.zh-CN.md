# 游戏规则简介

这是一次**严格写实**的刑侦调查。  
你将以侦查者的身份进入案件，通过**现场勘查、人物询问、线索追踪、时间线重建与矛盾整理**推进真相，而不是依靠直觉、巧合或“剧情便利”。

在本次游戏中，请牢记以下原则：

- **一切判断都必须建立在已获取的信息之上。**
- **不存在超自然、神迹、异术或凭空出现的关键证据。**
- **错误不会被直接宣告**，而会通过证词冲突、物证不符、时间延误、人物警觉与调查受阻逐步显现。
- **风险真实存在**：你问错人、信错话、惊动不该惊动的势力，都可能让线索消失，甚至让你自己陷入被动。
- 当你尝试指认、推断或定性时，裁定依据将是**证据链是否闭合**，而不是猜测是否“接近答案”。

如果调查一时停滞，你仍可继续从**尸体状态、现场痕迹、相关人物、物件来源、时间节点**等方向推进。  
记住：在这座皇城里，沉默、谎言、恐惧与自保，往往比刀更锋利。

# 背景故事概述

唐大和九年，深秋雨夜。  
长安城被连日阴雨浸得发冷，大明宫内外亦被无形的压力层层笼住。宰相与宦官集团角力不休，东宫与诸王彼此提防，朝局像一张绷到极限的弓，任何一点异动，都可能引发难以收拾的后果。

偏偏就在这样的时刻，素有“贤王”之名的**景王李涵**，死在了自己府中的书房里。

案发情形令人不安：  
书房门窗自内反锁，钥匙仍在死者腰间；景王倒在书案旁，胸口插着一柄本应陈列于墙上的**金柄匕首**；他的手中，还死死攥着半张写有“**甘露**”二字的绢纸；而指甲缝里，则残留着不属于书房的细微**银粉**。

表面看去，这像是一场密室中的自尽。  
可只要稍作审视，便会发现每一处细节都在彼此抵触。

更危险的是，景王之死绝不只是府中命案那么简单。  
它可能牵连私怨，可能关乎灭口，也可能只是更大风波掀开的第一道缝隙。你所面对的，不只是一个死者留下的谜团，还有活人出于利益、恐惧、忠诚与野心织成的沉默之网。

你不是局外人。  
从此刻起，你已被卷入这场风暴中心。

# 游戏目的

你的目标是以现实刑侦逻辑推进调查，在有限时间与不断升级的风险中，逐步完成以下事项：

1. **勘查景王书房与相关场所**，确认现场是否存在伪装、误导或遗漏痕迹。  
2. **核对死者状态、物件来源与时间节点**，重建景王死前的行动链。  
3. **询问相关人物并辨别证言真伪**，找出其中的隐瞒、回避与利益关联。  
4. **整理“密室”“匕首”“甘露”“银粉”等线索之间的关系**，判断哪些是核心，哪些是烟幕。  
5. 在证据足够前，谨慎做出你的阶段性判断，并决定接下来该继续深挖、转向旁证，还是优先自保。

你要追索的不是表面的解释，而是**能够经得起现实检验的真相**。  
但请记住：在皇城之中，查明真相从来不等于安全。

# 角色确认

你将扮演一名**被卷入景王之死核心事件的侦查者**。  
你不是单纯的旁观者，也不是无所不能的裁决者；你是这场调查的执行者、见证者，也是可能被各方试探、拉拢、利用或清除的人。

你的能力侧重将体现在以下方向：

- **观察**：发现现场细节、异常痕迹与物证矛盾。
- **交涉**：与证人、侍从、权贵或内侍周旋，从有限回应中撬出信息。
- **逻辑**：整理证据链，重建时间线，识别说法中的断裂与伪装。
- **追踪**：沿着物件、人物关系与行动路径继续深挖。

在正式开始前，请确认你的侦查者风格。你可以直接用一句话回答你的角色倾向，例如：

- 偏重观察与细节，不轻易相信口供  
- 擅长交涉与试探，从人心破绽入手  
- 注重逻辑与时间线，优先排查矛盾  
- 长于追踪与延伸调查，喜欢顺线摸到底  

**请现在确认你的角色调查风格，并准备进入案发现场。**
