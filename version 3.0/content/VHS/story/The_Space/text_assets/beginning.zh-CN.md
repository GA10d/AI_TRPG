# 游戏规则简介

你将进入一个几乎与现实无异的世界——至少表面上如此。城市、乡镇、森林、废墟、学校、旅馆、营地，它们仍然遵循你熟悉的常识与秩序，大多数人的生活照常运转。

但在某些角落，确实存在一些不该存在的东西：幽灵、执念、怪物、诅咒、杀人狂，或者某种无法被轻易理解的异变。

这些恐怖不会一开始就完整显露。它们通常先以细小、暧昧、不确定的方式侵扰你的感知——一段失真的录音、一扇自己打开的门、镜子里多出来的人影、夜路尽头一闪而过的轮廓、村庄里所有人都不愿提起的名字。随着你一步步靠近真相，那些潜伏的东西会变得越来越具体、越来越直接。

在这个世界里，恐怖不是舞台背景，而是一种会自然蔓延的现实。

这里没有数值面板，没有骰子声，没有显性的属性检定。你的每一次选择、每一句对话、每一次奔跑或屏息，都会通过环境反馈、感官细节和角色状态的改变来呈现结果。你越是靠近恐惧，恐惧也越容易靠近你——它会渗透进你的判断，侵蚀你的冷静，甚至影响你能否稳稳握住手边的工具。

这是一盘被反复覆盖、却始终没有彻底抹掉内容的旧录像。有些东西已经结束了。有些东西只是看上去结束了。

# 背景故事概述

地球在你脚下四百公里处，缓慢旋转。白色云层铺过海洋，城市灯火在夜侧像一把撒开的金粉。

你是“曙光号国际轨道站”延寿维护任务组成员之一。

这不是你们第一次一起执行任务，也不会是最危险的一次——至少现在看起来不是。

这里的人彼此熟悉到有些过分：谁会偷藏咖啡包，谁睡觉时会把固定带扣反，谁在跟地面通话前一定要整理头发，谁又会在早餐时讲一个只有自己觉得好笑的冷笑话。

今天的任务本该很普通。

右舷外部散热回路读数异常，太阳能阵列的一个旋转关节轻微卡滞，材料暴露实验架附近的传感器传回几组不稳定数据。地面控制中心建议安排一次短时舱外维修。窗口不长，但风险可控。你需要带一名队友出舱，完成检查、修复和校准。

站内的气氛甚至有些轻松。

妮娅说，如果这次维修顺利，今晚可以提前半小时开“电影夜”。米格尔已经开始争论到底该看老科幻片还是地面传来的球赛回放。艾莎提醒所有人先吃完早餐再谈娱乐计划。佐藤健司一边检查数据，一边小声说电影夜的投票程序根本不公平。叶莲娜没有参与争论，只是把舱外任务清单推到你面前，说：

*“先把站修好，再讨论人类文明的娱乐堕落。”*

你们笑了一阵。

然后警报灯重新闪了一下。

这一次，它没有立刻熄灭。

# 游戏目的

你的目标是完成舱外维修任务，将轨道站恢复至正常状态，与同组队员们一同度过这段驻留周期——并确保所有人安全返回。

调查系统中出现的一切异常。留意细微的变化：身体的不适、说话语气的偏差、不该出现在记忆中的对话、发生在你视线之外的接触。

在这个轨道站上，你最珍贵的资源不是氧气，不是电力，而是信任。

保护好它。

# 角色确认

你是“曙光号国际轨道站”延寿维护任务组成员。在任务组内，你拥有明确的专业身份。

请确认你的角色定位，或从以下建议方向中选择一个并进行补充描述：

- 舱外维修专家
- 轨道机械工程师
- 生命保障系统工程师
- 医疗官或生物实验载荷负责人
- 任务指令官候补
- 通讯 / 导航 / 对接系统专家
- 军方或航天机构背景的应急任务成员
- 私营航天承包方派驻专家

你叫什么名字？你的专业背景是怎样的？在这次任务中，你与哪位队员最熟悉，又与谁之间有过旧日摩擦？

**请描述你的角色，然后我们将正式开始。**
