# 《老宅狐影》重构版 / 《余庆堂白狐哭》

## 剧本必须项

### 1. 剧本元信息

**剧本名**：老宅狐影 / 《余庆堂白狐哭》  
**剧本 ID / 版本号**：THE_FOX_REBUILD_V1  
**题材标签**：中式恐怖、民俗悬疑、宅邸探索、奇幻哀剧、情感对质、生存压迫  
**风格标签**：压抑、阴湿、写实民俗、哀艳、慢性恐惧、高风险、伦理冲突  
**适用人数**：单人  
**角色创建方式**：自由创建；固定前提为“玩家因一封匿名来信，被引至即将拆毁的余庆堂老宅处理异常事件”  
**推荐流程长度**：中篇  
**推荐节奏**：前慢后紧，前段以宅院异象和人情疑云推进，中段逐步拼出前尘，后段在卯时逼近中完成真相与抉择收束  
**推荐主持风格**：重氛围、强引导、隐藏判定、允许轻度保护性兜底，但不替玩家完成关键道德判断  
**内容边界提示**：含民俗献祭、溺亡、活人封镇、幻觉诱导、家族压迫、旧式婚仪恐怖、角色自缢痕迹、情感欺骗、身份剥夺、可能的牺牲与开放性坏结局

---

### 2. 故事 Intro（用于显示）

你收到一封没有署名的旧信，信封里夹着一张快要脆裂的黑白照片：深山里的徽派老宅“余庆堂”，门楣下立着一名穿白衣的女子，脸被水痕晕开，只能看清她发间少了一半簪子。

信里只写了两句话：

**“宅子天亮后就要开拆。若再迟一夜，她就永远出不去了。”**  
**“别信牌位，先找井，再找镜。”**

黄昏时分，你抵达山中。暴雨截断了下山的路，老宅门前挂着尚未撕下的白灯笼，照出斑驳门匾与被红漆画过“拆”字的院墙。迎你进门的是余家最后的老管家周伯。他说自己已经等了很久，也说你最好别在宅里叫那位“白娘子”的名字。

但这座宅子里最危险的，不只是一个传说中的狐妖。

余庆堂真正可怕的，是一桩被余家压了近百年的旧案：

* 为什么前厅供着“狐仙牌位”，西厢房却像一间停在婚夜的灵堂？
* 为什么井里照出来的不只是你的影子，还有另一个女人被拖入水中的瞬间？
* 为什么那位被村人口口相传为“迷人吸阳”的白狐，留下的却是针线、家书、碎簪和一个始终没能兑现的承诺？
* 又为什么周伯明明像是来帮你的，却始终不肯先说出她真正的名字？

今夜，你要在旧镜、古井、家祠和阁楼之间，拼出一个女人的两段人生：她前世如何被献给这口井，今生又如何以狐身回到余家，最后被爱、家法与体面一起埋进水底。  
在卯时三刻之前，你必须做出决定：

**把她还给山里，把她重新镇回井下，还是成为余家谎言的新一部分。**

### 3. 玩家角色定位

**玩家身份**：自由创建

**建议身份方向**：

* 民俗调查者 / 写作者
* 受匿名委托前来处理怪事的人
* 建筑保护或拆迁相关人员
* 余家旧事的旁系知情者 / 旧友后人
* 对山中传说、狐仙信仰或民间异闻有长期兴趣的人
* 因私人创伤而对“未能被说出口的真相”高度敏感的人

**身份标签**：外来见证者 / 被选中的听故事者 / 可能成为“还名者”或“新镇物”的人  
**社会位置**：默认不属于余家核心血脉，但允许与余家旧闻、地方志、拆迁项目或山中民俗有弱关联  
**初始处境**：暴雨封山，玩家被迫在余庆堂内过夜，并迅速卷入一场围绕狐影、旧井、婚仪残迹与家族罪行展开的调查  
**初始认知范围**：

* 知道余庆堂是山里的老宅，即将被拆
* 知道当地流传“白娘子索命”的说法
* 知道周伯主动写信把自己引来
* 不知道“白娘子”是否真是狐妖
* 不知道她与余家、古井、断簪之间的完整关系
* 不知道周伯为何既像帮手，又像在拖延某件事

**初始目标**：
活着熬过这一夜，查明老宅异象的源头，决定如何处理井下之物，并设法在天亮前离开

**初始限制**：

* 山路被暴雨冲断，无法离山
* 老宅空间会随夜深逐渐错位
* 镜面、井水与旧画像可能主动篡改玩家感知
* 周伯不会一次性说出全部真相
* 一部分关键线索只会在高压或特定时辰显现

**初始优势**：

* 玩家是外人，不受余家“不能直呼其名”的旧规完全束缚
* 可自带职业技能、知识背景或调查工具
* 因不属于旧案中的任何一方，更有机会在“镇压 / 和解 / 说谎”三者之间做出独立选择

**是否固定主角**：否  
**是否允许自定义背景**：是  
**是否有预设关系网**：默认无；可在角色创建时选择与“余家旧闻”“地方民俗”“匿名来信”建立弱联系  
**与核心事件的关系**：被卷入余家百年旧债的外来旁观者，逐步成长为有资格给死者正名的人  
**与关键 NPC 的初始关系**：

* 与周伯：初见即合作，但对方明显有所隐瞒
* 与白娘子：未正式见面，但已通过信件、照片与宅中异象开始“挑选”玩家
* 与余砚生：未见面，但其留下的旧物与未尽承诺会逐步与玩家形成镜像关系

**玩家知道什么 / 不知道什么**：  
知道这不是一场普通闹鬼；不知道最可怕的并非狐妖，而是余家为了保住体面和家业，把一个名字、一段情意与两次死亡都改写成了“狐祟作乱”。

### 4. 核心目标

**显性目标**：  
探索余庆堂，查明狐影与古井异象的来源，处理白娘子的威胁，并在卯时三刻前活着离开。

**隐性目标**：  
找回“白娘子”被夺走的真实姓名与两段被余家抹去的生平，使她不再只是传说中的“妖”，而是一个被献祭、被利用、被迟到承诺害死的人。

**阶段目标**：

1. 进入余庆堂并与周伯形成基础合作
2. 在西厢房取得断簪左半，确认“白娘子”曾以人的身份生活在此
3. 在东书房与家祠取得余家旧档，发现“狐仙保宅”说法与实际记录存在冲突
4. 在天井与古井处建立“井新娘 / 镇狐”两条疑点
5. 进入阁楼，取得断簪右半、余砚生遗书与关于婚夜真相的关键片段
6. 进入井下封室，拼出白娘子的前世与今生为何都与这口井有关
7. 决定是否为她还名、是否揭示余砚生的迟疑、是否相信周伯最后的请求
8. 在卯时终局中完成镇压、和解、献祭或逃离中的一种选择

**成功条件**：

* 玩家形成一条完整且自洽的真相链
* 至少处理掉“名字 / 封印 / 情债”三者中的核心矛盾
* 至少达成一个成立的结局
* 玩家没有被老宅变成新的镇物或替身

**失败条件**：

* 未进入终局即迎来卯时三刻，白娘子完全实体化
* 玩家被井下拖拽并失去主动性
* 关键真相链被毁坏到无法形成有效判断
* 玩家主动接受镜中婚仪，成为余庆堂新的“守宅人”

**不可逆损失条件**：  
以下条件不是直接断本，而是**结局降级 / 路线锁死条件**：

* 断簪任一半彻底损毁
* 余砚生遗书被烧毁或遗失
* 周伯在终局前死亡或彻底与玩家决裂
* 牌位上的真名被再次抹去且玩家未记住
* 玩家恐惧与幻觉失控达到临界后未能恢复

### 5. 剧本主推进轴

**主推进轴**：探索调查 + 时间压力 + 情感揭示 + 生存风险  
**次推进轴**：民俗解谜 + 信任判断 + 幻觉识别 + 仪式抉择

这部剧真正推动故事的，不是“狐妖会不会扑上来”，而是四个不断收紧的问题：

* 她究竟是妖，还是先被人变成了“妖”的受害者？
* 余家供奉的是保家仙，还是被镇压后仍被榨干灵性的看门鬼？
* 余砚生留下的，到底是情深不寿，还是迟到一步的懦弱？
* 玩家在知晓全部真相后，会选择说真话、说安慰人的谎，还是延续余家祖祖辈辈最擅长的沉默？

这不是一个“驱鬼”故事。  
这是一个关于**给名字复位、给哀怨定形、给体面剥皮**的故事。

### 6. 场景 / 节点结构

#### 节点 1：门楼与前厅

**节点类型**：开场 / 信息枢纽  
**节点功能**：建立余庆堂整体气质，引入周伯、狐仙牌位与匿名来信，投放第一层误导  
**场景描述**：  
雨水顺着粉墙黛瓦往下淌，余庆堂的门槛比寻常人家的高，像一道刻意把外界隔开的旧规矩。前厅供桌上摆着褪色的狐仙牌位，香炉里灰烬潮得结块，牌位前却供着一对几乎像婚礼才会用的白蜡。门后还贴着拆迁办留下的红纸，红白并置，像喜事与丧事绑在了一起。

周伯提着煤油灯迎你进门。他极瘦，背驼得像被这宅子压了半辈子。说话客气，却总在你要追问时把目光移向院子中央那口井。

**可见元素**：

* 狐仙牌位
* 白蜡与香炉
* 门后拆字红纸
* 老旧族规牌匾
* 潮湿脚印
* 周伯的煤油灯与一串乌木钥匙

**可交互对象**：

* 牌位正反面
* 供桌抽屉
* 周伯
* 匿名来信
* 门后红纸
* 墙上的余氏家训

**可获得信息**：

* 余庆堂将在天亮后正式开拆
* 村里人把井下之物称作“白娘子”，却很少有人说出她原本是谁
* 牌位背面被新近刮擦过，像有人故意抹掉过字
* 余家祖训里有一句：“夜过子时，不照西镜，不唤井名。”
* 周伯不愿让玩家先去天井，反而有意把人往西厢房引

**可触发事件**：

* 供桌下掉出半张旧照片，照的是一名穿嫁衣的女子坐在西厢房镜前
* 玩家若直接触碰牌位，会感到木牌背后像有指甲刮擦
* 周伯说到“白娘子”时，前厅所有白蜡火焰会同时向井口方向偏斜

**进入条件**：默认开场  
**离开条件**：决定先去西厢房、书房或直接探查天井  
**节点风险**：低  
**指向下一步的钩子**：

* 周伯提示“镜子比井更早说话”
* 照片背后写着“簪断，人不归”
* 牌位后壁的刮痕像一个“白”字残笔

---

#### 节点 2：西厢房

**节点类型**：探索 / 第一惊吓 / 情感锚点  
**节点功能**：让白娘子首次具象化，建立“她曾在此像一个人一样活过”的质感  
**场景描述**：  
西厢房不像妖物巢穴，更像一场婚礼在最后一刻被人按停。拔步床上挂着落满灰尘的红盖头，镜台前的绣墩侧翻在地，梳妆匣里还有干裂的香粉和一支缺了半截的簪套。裂镜被一块旧红绸半掩着，像有人至死都不愿再看见自己的脸。

潮气里混着极淡的栀子香。那不是宅子发霉会有的味道，更像某个人曾经反复在这里梳妆、缝衣、等人。

**可见元素**：

* 拔步床与红盖头
* 裂镜
* 梳妆匣
* 绣了一半的嫁衣衣襟
* 断簪左半的金属反光
* 镜框暗格

**可交互对象**：

* 裂镜
* 床板与床幔
* 梳妆匣夹层
* 嫁衣
* 镜框背板

**可获得信息**：

* 白娘子并非只以“狐妖”形态存在，这里有明确的人类生活痕迹
* 嫁衣内侧绣着一个被线头反复拆改的字，最初像是“栀”
* 梳妆匣里藏着断簪左半，断口平整，不像岁月风化，更像被当场折断
* 镜框内侧留有细字：“若你回来，先别看井。”

**可触发事件**：

* 触碰裂镜时，触发**第一惊吓**：镜中女子先于玩家转头，半张脸是温顺新娘，半张脸浮出白狐轮廓；她不扑上来，只看着你，像在确认你是不是又一个来迟的人
* 玩家短暂陷入镜魇，听见婚夜锣鼓和井边泼水声同时响起
* 裂镜表面渗出一行血字：**“他答应过会回来。”**

**进入条件**：从前厅可直接进入  
**离开条件**：取得断簪左半 / 被第一惊吓逼退 / 意识到镜中另有暗格  
**节点风险**：中  
**指向下一步的钩子**：

* 断簪显然还有另一半
* “别看井”的留言说明写字人与井下之事有直接关联
* 嫁衣针脚与前厅牌位用布风格一致，暗示“供奉”与“婚事”属于同一套安排

---

#### 节点 3：东书房与家祠

**节点类型**：调查 / 信息整合 / 责任链建立  
**节点功能**：把“狐祟传说”转化成“家族掩盖”，让余家历史责任浮出水面  
**场景描述**：  
东书房的木架因受潮微微变形，账本和地方志挤在一起，像许多年没人敢认真翻过。与书房相连的家祠则截然相反，打扫得反常干净。祖先牌位整整齐齐，唯有最末一层空着一格，仿佛某个人本该被放进来，却被刻意排除在家谱之外。

墙上挂着一幅余家祖像，画中老爷穿深色团花袍，手里把玩的不是玉佩，而是一面小铜镜。

**可见元素**：

* 族谱与账本
* 祠堂空位
* 旧婚书残页
* 《驱邪秘录》抄本
* 画像中的铜镜
* 一只锁住的小抽屉

**可交互对象**：

* 族谱
* 账房册页
* 小抽屉
* 祖像背板
* 《驱邪秘录》批注页

**可获得信息**：

* 光绪末年余家曾在大旱后修井，并由此发家
* 同年村中有一名叫“白栀”的少女在“祭井”后失踪，地方记录刻意含糊其辞
* 数十年后，余家少爷余砚生曾执意纳一名来历不明的白衣女子为妻，但婚礼当夜发生“妖祟作乱”
* 《驱邪秘录》边缘有私人批注：**“狐不食人，人食狐名。”**
* 小抽屉里锁着一份未盖印的“送神婚契”，内容不像驱邪，更像以婚礼名义进行的封镇仪式

**可触发事件**：

* 玩家将断簪靠近族谱时，空位处会慢慢浮出一行被墨覆盖过的旧字：“白栀”
* 若追问周伯过深，他会明显失态，第一次说出“她本来也不是想害人”
* 祖像中的铜镜会在无人碰触时偏转角度，映出通往天井的门缝

**进入条件**：自由可达  
**离开条件**：取得送神婚契 / 解开族谱疑点 / 被异象惊退  
**节点风险**：中  
**指向下一步的钩子**：

* “祭井”与“婚夜封镇”可能是同一悲剧的前世今生
* 余砚生是理解白娘子真相的关键
* 画像里的铜镜提示真正的封印核心不在书房，而在井与镜的联动处

---

#### 节点 4：天井与古井

**节点类型**：高压探索 / 民俗压力场 / 真假识别  
**节点功能**：建立“井下有两层旧债”，让白娘子的前世线正式成立  
**场景描述**：  
天井常年积水，四角石缸里鱼全翻了肚皮，白肚映着天光，像许多盯着人的眼。井口被铁链半圈着，链上挂了旧铜铃，风不大，铃却会自己动。井绳上缠着女人的长发，一缕缕早已与麻绳风干成一体。每当你低头看井，倒影总要慢半拍才跟上你。

这里像整座宅子真正的心脏，也是所有谎言绕不开的伤口。

**可见元素**：

* 古井与铁链
* 铜铃
* 缠发井绳
* 四角石缸
* 井沿刻痕
* 井壁渗水痕迹

**可交互对象**：

* 井水倒影
* 铁链与铜铃
* 井沿刻字
* 石缸死鱼
* 井绳长发

**可获得信息**：

* 井沿旧刻中残存“井娘入水，换一宅平安”的字句
* 若玩家对井中倒影说出“白娘子”，井水会翻涌；若说出“白栀”，井面会短暂静止
* 井水回声会分成两层：一层是少女求救，一层是狐兽低鸣，暗示她的怨并非单一来源
* 铜铃内侧刻着余家家纹，说明井下封镇不是外来道士偶发之举，而是余家主动设计

**可触发事件**：

* 玩家靠井过久，会触发“井手抓踝”事件，被拖入一段前世幻象：大旱之年，一个少女被按着头送到井边，手里还护着一只受伤小白狐
* 周伯若在场，会第一次明确阻止玩家再看井，并说出“她前世就死在这里”
* 井里会映出余砚生站在雨里的身影，却始终不肯踏进井沿一步

**进入条件**：默认可达，但建议在取得至少一条书房或西厢房线索后进入  
**离开条件**：确认“白栀”这个名字有效 / 被高压逼退 / 发现需要进入阁楼与井下  
**节点风险**：高  
**指向下一步的钩子**：

* 白娘子的“前世”与井娘献祭直接相关
* 余砚生与她的关系不是简单的人狐相恋，而是被前世旧债与今生情爱一起绞住
* 阁楼很可能藏着婚夜前后的最后证物

#### 节点 5：阁楼与绣楼夹层

**节点类型**：探索 / 幻象 / 真相转折  
**节点功能**：完成白娘子今生线与余砚生线的闭环  
**场景描述**：  
木梯陡而窄，像通往另一个时间层。阁楼尘土厚得像灰雪，梁上却有一道陈年的勒痕。这里原本应是储放旧书与嫁妆的地方，如今却同时保留着新郎的墨迹和野兽般的狐毛。最深处的夹层里放着一张半毁的《白娘子本相图》，画中的白衣女子面容端正，不似怪物，反而像被故意画成“妖”的人。

一只旧木匣压在石床般的木板上，匣子里藏着断簪右半、未寄出的书信与一小截褪色的红绸。

**可见元素**：

* 梁上的旧勒痕
* 狐毛与旧布
* 木匣
* 《白娘子本相图》
* 余砚生手稿
* 断簪右半

**可交互对象**：

* 木匣
* 手稿与遗书
* 画轴
* 梁柱
* 夹层地板

**可获得信息**：

* 余砚生确实爱她，并知道她真正的名字叫“白栀”
* 白栀并非来余家吸阳，她是循着前世井边残魂而来，借狐身化形，只想查清自己为何总梦见自己死在这宅里
* 她与余砚生曾约定私奔并公开余家旧账；断簪就是约定信物
* 婚夜前，余砚生承诺“去拿证据和钥匙，很快回来”，但他在母亲哀求与家族压力前迟疑了
* 他赶回来时，封井已成；他后来在阁楼自缢，留下遗书：**“不是我没爱她，是我先去救了别的东西。”**

**可触发事件**：

* 取得断簪右半会触发完整婚夜幻象：白栀被迫穿上嫁衣，断簪，镜封，入井；余砚生在雨中跑来，却只看见被盖上的井板
* 玩家若触碰梁上的勒痕，会听见余砚生最后一句话：“我来迟了。”
* 若玩家已对周伯建立信任，周伯会承认自己的外祖母阿满曾是白栀在余家时唯一愿意帮她的人

**进入条件**：从天井或书房线索指向，或自由探索到达  
**离开条件**：拿到断簪右半 / 读完关键遗书 / 因幻象压力撤退  
**节点风险**：高  
**指向下一步的钩子**：

* 断簪集齐后可开启“还名 / 安抚”路径
* 《白娘子本相图》与《驱邪秘录》可开启“镇压 / 焚毁”路径
* 夹层地板下方传来的水声，指向井下封室入口

#### 节点 6：井下封室

**节点类型**：真相中心 / 高风险探索 / 终局前置  
**节点功能**：给出最硬核的物证，让“前世献井 + 今生封狐”两层悲剧彻底闭合  
**场景描述**：  
井下不是单纯的水井，而是一间被水脉半淹的封室。四面嵌着八面旧铜镜，镜面大多发黑，只剩中间一面仍能照清人的脸。地上散着断裂符绳、婚仪用的银铃、一对早已锈蚀的脚镯，以及两种纠缠在一起的遗骸痕迹：一具女性人骨残段，一副细小白狐骨架。

它们不是并排放着，而是像死前曾紧紧抱在一起。

**可见元素**：

* 八面镇狐铜镜
* 女性遗骸与白狐骨架
* 银铃与脚镯
* 封井石板内侧符咒
* 一块被水泡烂的木牌残片
* 一串被周伯藏过又拿回的井钥匙

**可交互对象**：

* 中央铜镜
* 遗骸
* 木牌残片
* 符绳
* 井钥匙

**可获得信息**：

* 白栀前世确实以“井娘”身份死于此处
* 后来救过她残魂的白狐，也在婚夜被一并封杀于此，因此“白娘子”既是狐妖，也是背着白栀名字活下去的第二条命
* 余家并非“斩妖”，而是把一个知道前世真相、又准备揭穿今生旧账的存在重新沉井
* 木牌残片上写的不是“狐仙”，而是“白栀灵位”
* 周伯其实一直保留着开封室的钥匙，却不敢先开，因为一旦真相被确认，他就再也不能替余家用“妖”字遮羞

**可触发事件**：

* 白娘子在中央铜镜中完全显形：不再只有狐面或新娘脸，而是同时呈现少女、狐灵与井中尸影三重重叠姿态
* 她会向玩家提出第一次明确诉求：**“把我的名字还给我，还是也叫我一声妖？”**
* 若玩家此时展示断簪，她敌意会短暂下降；若展示《驱邪秘录》或点燃符纸，她会立即转入暴怒状态

**进入条件**：建议至少取得断簪任一半 + 一条书房线 + 一条阁楼线后进入  
**离开条件**：触发终局准备 / 带走木牌残片 / 被白娘子驱逐  
**节点风险**：极高  
**指向下一步的钩子**：

* 木牌残片与断簪可进入“还名归山”路径
* 符纸、朱砂笔与本相图可进入“血墨镇狐”路径
* 若玩家对余砚生真相做了选择性隐瞒，可进入“井底未嫁”或“替身新郎”路径

---

#### 节点 7：卯时终局（天井 / 家祠）

**节点类型**：收束 / 仪式 / 结局  
**节点功能**：让玩家决定真相应该如何被说出、被安慰、被再次利用，或被彻底埋掉  
**场景描述**：  
卯时前后的余庆堂像在缓慢苏醒。井水开始漫过天井砖缝，家祠的牌位自行轻响，白蜡同时流泪。整个宅子像一具终于想起自己死因的身体。

终局空间分为两类：

**A. 天井终局**  
更接近还名、下井、归山、履行承诺。这里处理的是“她是谁”和“她是否还能离开”。

**B. 家祠终局**  
更接近镇压、焚毁、改写、让余家牌位继续压过她。这里处理的是“余家该不该付代价”和“玩家愿不愿继续说谎”。

**可触发事件**：

* 周伯彻底坦白或彻底崩溃
* 白娘子要求玩家念出木牌上的真名
* 余砚生的残影会在关键时刻出现，但不会替玩家作任何选择
* 井水上涨、铜铃齐鸣、镜面全部映出婚夜幻象
* 玩家必须决定：是否归还断簪、是否出示遗书、是否说出余砚生来迟的真相、是否用朱砂笔完成镇灵符

**进入条件**：

* 任一终局可达：玩家已进入井下封室并获得足够判断材料
* 完整终局：断簪集齐 + 真名确认 + 余砚生遗书在手 + 木牌残片在手
* 强制坏结局：卯时三刻自动到来时尚未完成选择

**离开条件**：进入结局  
**节点风险**：极高  
**指向下一步的钩子**：直接进入结局结构

### 7. 关键实体

#### 白栀 / “白娘子”

**类型**：敌对者 / 受害者 / 场景核心灵体  
**表层印象**：白衣、长发、半边狐面、发间断簪，新娘与兽影交替出现；既会温柔地引人，也会在被叫错名字时立刻翻脸  
**真实动机**：她真正想要的不是单纯索命，而是拿回被余家连续夺走两次的“身份”。第一次，她前世作为井娘白栀被献井；第二次，狐灵背着白栀的名字活成了人，却又在婚夜被打回“妖”的位置。  
**立场**：对继续把她当“狐祟”“镇宅物”的人高度敌对；对愿意倾听并承认真名与真相的人存在被安抚的可能  
**与玩家关系**：把玩家视作新的见证者，也可能视作第三个来迟的人  
**知情范围**：知道自己两次死在这宅里的经过；不知道余砚生到底是纯粹背叛了她，还是在迟疑与愧疚里一起毁掉了自己  
**对玩家隐瞒的信息**：她最恨的不只是余家，也恨“迟到的承诺”；若玩家只给她安慰性的谎而不给她真名，她会把那视作另一种侮辱  
**可提供的信息 / 资源**：婚夜幻象、井中回声、真名反应、引导玩家找到井下封室  
**会制造的阻碍**：镜魇、空间错位、拖拽入井、伪装成周伯或旧日新娘、逼迫玩家参加虚假的婚仪  
**行为触发逻辑**：被叫成“狐妖”、看见符纸、玩家毁坏断簪、玩家重复余家口径、玩家在关键时刻选择沉默  
**变化条件**：断簪集齐、真名被说出、遗书出示、余砚生真相被正面承认  
**退场条件**：被还名归山 / 被重新镇压 / 与玩家缔成替代性婚约 / 因真相被掩埋而继续留宅

---

#### 周伯

**类型**：同伴 / 历史看守人 / 潜在误导者  
**表层印象**：瘦削、谨慎、礼数周全，像个想把整件事平平稳稳“处理掉”的老管家  
**真实动机**：他既想让白栀离开，又怕真相彻底掀开后余家最后一点体面和自己守了一辈子的意义一起塌掉。他不是恶人，但长期用“替死人找个说法”的方式替活人遮羞。  
**立场**：摇摆；前期偏向保宅和止事，后期视玩家行为可能转向坦白或继续遮掩  
**与玩家关系**：最早的向导与信息源，也是最重要的不可靠叙述者  
**知情范围**：知道井下封室、白栀真名、外祖母阿满与婚夜旧事、余家祖先主动封镇的事实  
**对玩家隐瞒的信息**：匿名信是他写的；井钥匙一直在他身上；他母亲一支其实是余家旁支与婢女生下的后裔，因此他对余家既有主仆忠诚，又有血脉羞耻  
**可提供的信息 / 资源**：钥匙、路线、祖宅规矩、最后的口供、可能的终局协助  
**会制造的阻碍**：拖延、粉饰、劝玩家“别问那么真”、在危险场景强行把玩家拉开  
**行为触发逻辑**：玩家是否尊重他、是否搜集到足够证据、是否表现出把白栀当“纯妖物”或“纯可怜人”的极端倾向  
**变化条件**：玩家是否出示证据而非空口逼问、是否在终局前仍给他坦白机会  
**退场条件**：协助完成终局 / 为护玩家或白栀而死 / 因恐惧彻底逃离 / 留在老宅与余家同亡

---

#### 余砚生

**类型**：缺席关键实体 / 关系对象 / 历史幽影  
**表层印象**：只通过遗书、手稿、幻象与勒痕出现，是一个看似深情、实则极不完整的人  
**真实动机**：想爱白栀，想救她，也想揭开余家旧账，但他最致命的缺陷是总想“先把别的事也处理好”，于是最终对谁都来迟了一步  
**立场**：情感上站在白栀这一边，行动上却没能真正脱离余家结构  
**与玩家关系**：玩家越了解他，越会被逼着判断——迟到的爱，还算不算爱  
**知情范围**：知道白栀真名、知道井娘旧案、知道送神婚契是假的驱邪真封镇  
**对玩家隐瞒的信息**：他不是单纯被阻拦，而是在关键一刻确实犹豫过  
**可提供的信息 / 资源**：遗书、断簪右半、婚夜真相、安抚或激怒白栀的关键话语  
**会制造的阻碍**：其“看似浪漫”的形象会误导玩家，把整件事误读成单纯苦恋故事  
**行为触发逻辑**：玩家触碰断簪、读遗书、接近阁楼梁痕、在终局中谈及承诺  
**变化条件**：玩家是否让他的真实迟疑被承认，还是选择替他粉饰  
**退场条件**：真相被完整说出后消散 / 继续作为婚夜残影循环 / 被玩家作为美化叙事的一部分利用

---

#### 余绍廷（余家老爷）

**类型**：历史责任人 / 退场实体  
**表层印象**：只存在于画像、账本、祠堂规矩与仪式遗留中，是一个极重体面与家业的人  
**真实动机**：把一切超自然力量、女性牺牲与家中丑闻都转化为可供家族使用的资源，只要宅子稳、名声在，就可以牺牲任何人或任何存在  
**立场**：责任链第一环，也是“妖化受害者”的主导者  
**作用**：把前世献井与今生封狐联系成一套连续的家族暴力逻辑

---

#### 余庆堂老宅

**类型**：场景意识体 / 记忆容器  
**表层印象**：潮湿、守旧、结构精巧，仿佛每一面墙都在故意留住声音与影子  
**真实动机**：它不是活物，却长期被井水、镜阵、牌位和家规喂养成了会“选择性记忆”的空间——它保存真相，也帮助余家改写真相  
**立场**：中立偏恶；会放大压抑、沉默和重复的仪式  
**与玩家关系**：既是调查舞台，也是会主动误导玩家的巨大机关  
**可提供的信息 / 资源**：镜中复现、脚步回音、潮气中的旧字、空间重组带来的新路线  
**会制造的阻碍**：错路、回廊循环、影像错位、把同一事件以不同版本反复演给玩家看  
**变化条件**：时辰推进、井水上涨、真名被说出、牌位被毁  
**退场条件**：白栀被还名后逐步失去异状 / 被强行镇压后暂时恢复沉寂 / 在坏结局中继续作为“吃人旧宅”存在

### 8. 关键信息单元

### 10. 风险与后果

**核心风险源**：

* 白栀的镜魇与拖拽
* 井水上涨与空间错位
* 周伯的隐瞒和迟疑
* 玩家误把悲剧读成单纯人狐恋或单纯斩妖任务
* 时辰推进
* 在真相不完整时仓促进行仪式

**风险升级方式**：

* 反复照镜
* 长时间停留井边
* 粗暴逼问周伯
* 毁坏断簪、木牌或遗书
* 在没有足够证据前下结论
* 拖到寅时后仍未进入井下封室

**即时后果**：

* 幻觉、错路、短暂失温
* 被井水拖拽、腿脚受伤
* 白栀敌意上升
* 周伯拒绝合作或交钥匙
* 某些场景暂时封闭

**延迟后果**：

* 安抚路径失败
* 周伯不再愿意说出全部真相
* 玩家开始分不清前世幻象与当前现实
* 白栀把玩家视为“又一个只会说好听话的人”

**隐性后果**：

* 玩家越执着于“解决怪物”，越容易重演余家把她物化的逻辑
* 玩家若只想“安慰她”，却不肯承认真相，同样会招致更激烈反噬
* 玩家若替余砚生美化过度，容易导向替身婚约或不完整安抚

**长期后果**：

* 真相再次被妖化处理
* 余家责任被继续转嫁给“狐祟”
* 玩家成为新一轮地方传说的一部分
* 老宅在开放坏结局中转化为更危险的“吃人景点”或“被供奉的凶宅”

**对剧情推进的影响**：  
风险不阻止推进，但会让每一步推进付出更高的代价：更多幻象、更少时间、更差的关系、更窄的结局空间。

**对结局的影响**：  
直接决定玩家能否达成完整真相、是否只得到美化过的安慰、是否必须通过牺牲或镇压求生，以及是否会落入替身婚约与轮回类坏结局。

### 11. 分支点

#### 分支点 1：先信周伯的规矩，还是先信宅子的异象

**触发时机**：开场后  
**玩家可做选择**：

* 先按周伯建议去西厢房
* 直接调查天井古井
* 自主绕开周伯去书房翻旧档

**直接后果**：

* 先去西厢房：更早建立情感线与断簪线
* 先去古井：更早建立前世线与高压风险
* 先去书房：更早建立责任链，但白栀更晚具象

**延迟后果**：

* 先走情感线更利于和解结局
* 先走井线更利于快速进入真相中心，但恐惧压力更高
* 先走书房更利于理性判断，但更易把白栀初看成“案件对象”而非受害者

**影响对象**：白栀、周伯、后续节点节奏  
**是否可回头**：可  
**是否为隐性分支**：否

#### 分支点 2：是否当面对质周伯

**触发时机**：取得两条以上关键物证后  
**玩家可做选择**：

* 立刻摊牌
* 暂时不说，继续调查
* 以试探方式引导他自己坦白

**直接后果**：

* 摊牌：可能快速拿到钥匙，也可能让周伯封口
* 不说：可保留合作表面，但终局前压力更大
* 引导：最稳妥，但耗时

**延迟后果**：

* 逼得太急会让周伯在终局时偏向“快点镇了她”
* 拖得太久则可能错过他主动协助的窗口

**影响对象**：周伯、钥匙、井下封室可达性  
**是否可回头**：部分不可  
**是否为隐性分支**：是

#### 分支点 3：如何理解余砚生

**触发时机**：取得遗书后  
**玩家可做选择**：

* 认为他是深情而无力的人
* 认为他是爱却迟疑、因此同样有责的人
* 认为他本质上也是余家的一部分

**直接后果**：

* 偏浪漫理解：更容易进入安抚或替身婚约
* 平衡理解：最利于完整真相结局
* 偏严厉理解：更利于镇压或冷硬公开路线

**延迟后果**：

* 影响终局中玩家会不会把遗书完整念出
* 影响白栀是否接受“真话”或把玩家视为下一个敷衍她的人

**影响对象**：白栀、终局情绪基调、结局落点  
**是否可回头**：可，但代价会体现在对白与白栀态度中  
**是否为隐性分支**：是

#### 分支点 4：终局选择——还名、镇压、说谎还是代替

**触发时机**：进入卯时终局  
**玩家可做选择**：

* 归还断簪与木牌，为她还名
* 使用朱砂笔、符咒与本相图重新镇压
* 只给她一个好听的说法，不完整揭示真相
* 接受镜中婚仪，成为新的“承诺者”

**直接后果**：

* 还名：高风险但最完整
* 镇压：求生率高但道德代价重
* 说谎安抚：可能得到苦涩平息，也可能触发反噬
* 接受婚仪：进入坏结局或隐藏结局

**延迟后果**：  
决定余庆堂是结束、沉寂、继续吃人，还是把玩家一起写进它的历史里。

**是否可回头**：基本不可  
**是否为隐性分支**：否

### 12. 结局结构

#### 结局一：还名归山

**结局类型**：真相达成 / 生还 / 和解  
**达成条件**：

* 断簪集齐
* 真名“白栀”被完整确认
* 木牌残片在手
* 余砚生遗书未毁
* 玩家在天井或井边为她同时归还“名字、簪子与真相”

**必要前提**：责任链基本完整  
**禁止条件**：断簪全毁；真名未确认；玩家选择继续用“白娘子 / 狐妖”替代其身份  
**收束对象**：白栀、周伯、余砚生、玩家、余庆堂  
**最终揭示内容**：  
白栀不是被斩的妖，而是被余家两次抹去身份的人：前世是井娘，今生是带着她名字回来的狐灵。余砚生确实爱她，也确实来迟。只有当这两件事被一起承认，她才能真正离开。  
**主题落点**：真正的超度不是温柔，而是把名字和责任同时归位。  
**未完成支线如何收束**：周伯会活着看见宅子失去异状，余家牌位自行裂开；是否把此事向外界公开，留给后日谈处理。

#### 结局二：井底未嫁

**结局类型**：苦涩和解 / 半真相达成  
**达成条件**：

* 断簪集齐
* 玩家为她完成象征性归簪
* 但未完整说出余砚生迟疑，或刻意用较温柔的版本安抚她

**必要前提**：情感线成立  
**禁止条件**：断簪缺失；玩家全程将其视为纯妖  
**收束对象**：白栀、余砚生残影、玩家  
**最终揭示内容**：  
白栀接受了一个带保留的答案，怨气平了大半，宅中异状暂歇，但余家真正的羞耻并未被彻底揭开。她像新娘一样在井边最后一次低头，却终究没等来一场真正完成的婚礼。  
**主题落点**：有些创伤能被安慰，却没有被真正说明。  
**未完成支线如何收束**：周伯会感谢玩家“给了她一个好结局”，但这个说法本身仍带着余家式的粉饰。

#### 结局三：血墨镇狐

**结局类型**：生还 / 压制 / 黑暗胜利  
**达成条件**：

* 获得镇压法门
* 在家祠或井口完成镇灵符
* 焚毁本相图或以牌位替换木牌残片
* 玩家活着撑到白栀被重新封住

**必要前提**：至少取得一条仪式线索  
**禁止条件**：玩家已经完整为她还名  
**收束对象**：白栀、周伯、玩家、余庆堂  
**最终揭示内容**：  
玩家成功把白栀重新压回“狐祟”的位置，余家祖辈的谎再度获胜。宅子短期内安静下来，但井里仍会传出女人与狐共同呜咽的声音。  
**主题落点**：秩序可以恢复，正义未必会来。  
**未完成支线如何收束**：周伯可能存活，但会在余生继续守着秘密；若宅子后续被拆，诅咒可能外溢。

#### 结局四：替身新郎

**结局类型**：失败 / 关系异化 / 轮回  
**达成条件**：

* 玩家过度认同白栀，却没有给出完整真相
* 接受镜中婚仪、披上红绸或主动交出自己的名字
* 卯时三刻到来前未完成真正的还名或镇压

**必要前提**：白栀对玩家有较高“信任”或执念  
**禁止条件**：完整真名路径已成立  
**收束对象**：白栀、玩家、余庆堂  
**最终揭示内容**：  
玩家成了她终于等到的“回来的人”，但这不是救赎，而是余庆堂为延续自己选择的新镇物。下一个收到匿名信的人，会在黑白照片上看见你站在她旁边。  
**主题落点**：被悲剧打动，不等于能够承担悲剧。  
**未完成支线如何收束**：周伯可能逃走，也可能继续作为你的“管家”留在宅中。

#### 结局五：新牌位

**结局类型**：开放失败 / 社会性后患  
**达成条件**：

* 玩家未完成核心选择而强行离宅，或毁坏大量证物导致真相链断裂
* 白栀未被安抚也未被完全镇压

**必要前提**：真相残缺  
**禁止条件**：无  
**收束对象**：玩家、周伯、余庆堂、地方社会  
**最终揭示内容**：  
天亮后余庆堂照常拆除，村里只传言“老宅闹过一夜鬼”。几个月后，拆迁工地或新修民宿不断出事，前厅废墟上多出一块没有名字的新牌位。  
**主题落点**：被掩埋的东西，不会因为建筑消失就停止存在。  
**未完成支线如何收束**：玩家可能带走一部分证据，却永远分不清自己到底有没有真正离开那口井。

#### 结局六：隐藏结局·以妖为名

**结局类型**：动态 / 黑暗 / 反主题  
**达成条件**：

* 玩家主动站到余家一边，把所有真相重新定义为“狐妖惑人”
* 或利用白栀之力达成私人目的，如保住拆迁项目、求财、求名、求活

**最终揭示内容**：  
根据玩家行为生成后日谈：可能是玩家成了靠“驱狐”成名的人，可能是余家旧宅改成了打卡胜地，可能是新的“狐仙牌位”被重新供起，而木牌背后的真名又被刮掉一层。  
**主题落点**：把受害者重新命名成怪物，是最古老、也最有效的谋杀方式。

### 13. 主持约束 / Agent 约束

**是否允许强引导**：允许，在玩家停滞时通过镜面异象、井水回声、周伯的反常阻拦、族谱空位等方式推进  
**是否允许主动补关键线索**：允许，尤其是“白栀真名”“送神婚契”“断簪”“余砚生遗书”“木牌残片”  
**是否允许保护性兜底**：允许轻度兜底，避免一次误判直接断本  
**是否允许模糊判定**：允许，且推荐  
**是否允许压缩场景**：允许，但不得跳过以下体验：

* 西厢房第一惊吓
* 天井建立前世井娘线
* 阁楼取得余砚生真相
* 井下封室确认双重死亡
* 卯时终局抉择

**哪些信息不能提前说**：

* 白娘子的真名是白栀
* 她前世就是井娘
* 狐灵并非来作恶，而是背着白栀的名字回宅查真相
* 余砚生真正的过错不是不爱，而是来迟
* 周伯一直持有井钥匙

**哪些结论不能替玩家做**：

* 白栀该不该被原谅
* 余砚生算不算值得同情
* 周伯是否值得信任
* 最终应选择还名、镇压、说谎还是接受婚仪
* 是否向外界公开余家旧案

**是否需要确保某类体验成立**：  
必须确保：

* 井、镜、牌位三者形成统一恐怖意象
* “她曾是一个人”这层体验必须成立
* 余家活人的处理方式比狐妖本身更令人寒心
* 真相比怪谈更残忍
* 玩家的每一次推进都更接近一个“必须表态”的道德选择

**是否必须保留悬念**：必须  
**是否必须避免某类走向**：

* 不把白栀写成纯吃人狐妖
* 不把周伯写成单纯工具人或纸片叛徒
* 不把余砚生写成完美情圣
* 不把余家责任全都稀释成“时代如此”
* 不写死玩家职业和解决问题的方法

* **丑时（01:00）**：周伯梦游，井水上涨，古井线危险显著提升
* **寅时（03:00）**：井下封室入口开启，白栀可短时在无镜处显形
* **卯时三刻（05:45 左右）**：强制终局；若未完成处理则进入失败 / 坏结局判定

**倒计时**：有  
**时间节点事件**：有  
**错过窗口的后果**：

* 子时前未取得至少一条人物情感线，后续更易误判白栀性质
* 寅时前未进入井下封室，终局压力大幅抬高
* 卯时三刻前未完成核心选择，白栀完全实体化，出口失效

**白天 / 夜晚差异**：  
本剧默认从黄昏入夜，强调“最后一夜”的压迫感；白天只作为后日谈存在。

### 3. 特殊机制模块

**镜魇机制**：  
镜子不只是惊吓点，也是叙事装置。镜中会给出过去的片段，但不保证顺序正确；同一件事可能先以美化版、再以真实版出现。

**井听机制**：  
玩家若向井中说话、投物或照看倒影，会触发回声式线索。不同称呼会引出不同反应：叫“白娘子”更易触发攻击，叫“白栀”更易触发真相片段。

**还名机制**：  
本剧安抚并非靠单纯情感，而是要满足“真名 + 物证 + 责任承认”三项中的大部分。只讲感情不讲真相，无法达成完整救赎。

**替身婚仪机制**：  
若玩家在高压下多次选择安慰而回避真相，或主动接受红绸、铜铃、镜中新郎位，则会逐步被识别为“承诺补位者”。

### 5. 敌对系统 / 怪物系统

**敌对者名称**：白栀 / 俗称“白娘子”  
**威胁来源**：前世井娘惨死、今生狐身被封、真名遭抹去、婚约被断裂所叠加的复合怨念  
**行为模式**：

* 借镜现身，先诱导后逼视
* 通过井水倒影拖拽玩家入幻象
* 模仿周伯或温柔新娘语气试探玩家
* 在玩家说错名字、毁坏证物或重复余家话术时爆发攻击
* 在玩家接近完整真相时，主动补全最痛的那部分记忆

**弱点**：

* 真名“白栀”
* 完整断簪
* 木牌残片
* 对余家责任与余砚生迟疑的明确承认
* 不再被称作单纯“狐妖”

**出现场景**：西厢房、天井、阁楼、井下封室、任何可反光水面或镜面  
**升级条件**：照镜、拖延、毁证、井水上涨、卯时逼近  
**压迫方式**：先制造美化幻象，再逼玩家亲眼看见真相，最后要求玩家表态  
**退场方式**：还名归山 / 被重新镇压 / 与玩家缔成替身关系 / 继续滞留旧宅

### 6. 关系网络

**白栀（前世井娘） ↔ 白狐**：  
前世白栀在被献井前护住了受伤白狐；白狐承其残魂与名字修成人形，成为后来被称作“白娘子”的存在。

**白栀（今生狐灵） ↔ 余砚生**：  
真正相爱，但爱被家法、旧债与迟疑同时毁掉。两人的悲剧核心不是“不爱”，而是“爱得不够快、不够决绝”。

**白栀 ↔ 余绍廷 / 余家**：  
被前世献祭、今生再封镇，是连续的家族暴力受害者。

**周伯 ↔ 白栀**：  
既愧疚又畏惧；他外祖母阿满曾帮助白栀藏簪，因此周伯一直背着“该还她一个名字”的私人债。

**周伯 ↔ 余家**：  
忠诚与羞耻并存；他守宅，既是责任，也是无法离开的旧式主仆观念。

**玩家 ↔ 白栀**：  
从被试探的外人，逐步变成必须表态的见证者，也可能被她选为替代承诺者。

### 7. 误导信息 / 红鲱鱼表

**误导 1：白娘子是单纯吸人阳气的狐妖**

* 看似支持：狐面、井中拖拽、镜中诱惑
* 实际真相：她的攻击性来自被连续妖化与封镇后的复合怨念
* 如何纠正：嫁衣痕迹、真名线、井下双重遗骸
* 若玩家信了：更易走向血墨镇狐或隐藏坏结局

**误导 2：余砚生是纯粹痴情受害者**

* 看似支持：遗书、自缢、断簪信物
* 实际真相：他真心爱过，但关键时刻确实迟疑过
* 如何纠正：婚夜幻象、遗书原文、井边回声
* 若玩家信了：可能导向不完整安抚或替身婚约

**误导 3：周伯知道全部真相所以一定不可信**

* 看似支持：他写匿名信、藏钥匙、反复劝阻
* 实际真相：他是摇摆的守秘者，不是主谋
* 如何纠正：阿满支线、坦白事件、终局选择
* 若玩家信了：可能失去关键援助与井下入口

**误导 4：前世井娘与今生狐灵是两个完全无关的人**

* 看似支持：一人为人，一者为狐
* 实际真相：白狐承了白栀的残魂与名字，两段生命已经缠成一个存在
* 如何纠正：井下遗骸、真名机制、白栀自述
* 若玩家信了：无法理解她为何如此执着于“名字”而非单纯“爱情”

### 9. 示例流程

**标准调查回合示例**：  
前厅接触周伯与牌位 → 西厢房触发第一惊吓并取得断簪左半 → 书房确认送神婚契 → 天井建立井娘疑点

**关系推进示例**：  
玩家在天井被拖入前世幻象 → 周伯强行把人拉开 → 玩家选择追问或暂缓 → 决定周伯后续是继续敷衍还是松口

**节点切换示例**：  
阁楼拿到遗书与断簪右半 → 白栀敌意短暂平复 → 玩家意识到必须下井 → 井下封室开启终局前置

**提示递进示例**：  
最初只知道“宅里有狐妖” → 之后知道“她曾是新娘” → 再知道“她前世叫白栀，死于祭井” → 最终知道“今生狐灵承她名字回宅查真相，又被余家以婚礼之名再次沉井”

**结局收束示例**：  
若断簪齐备、真名确认、遗书在手、木牌残片在手，则进入还名归山；若只想快点止事并完成符咒，则进入血墨镇狐；若以温柔谎言替代完整真相，则进入井底未嫁或替身新郎。

### 10. 变体与扩展

**替换真相**：  
可把“前世井娘”替换为“冥婚陪葬”“山神祭品”“祠堂活封”等地方民俗变体，但首版建议保留“井娘 + 狐灵承魂”结构，以保证名字与井的意象高度统一。

**可替换角色**：  
周伯可改为中年女看宅人、地方志老师或拆迁联络员，但需保留“对余家既有债又有羞”的复杂性。

**可替换结局**：  
可增设“公开旧案后玩家成为地方异闻焦点”的媒体型后日谈，或“老宅拆除后白栀只剩名字存活”的都市怪谈结局。

**难度档位**：

* 低难：周伯更早坦白，镜魇误导减少
* 中难：标准版
* 高难：井水上涨更快、幻象更多、替身婚约更容易触发

**多周目变化**：

* 第二轮可提前让牌位背面的真名笔画显现
* 某些镜中片段会从美化版自动切换为真实版
* 隐藏结局可扩展为“玩家成为下一封匿名信的寄件人”

**单人 / 多人适配差异**：  
当前为单人版；若改多人，可把周伯的部分功能拆给玩家阵营中的“余家旁支后人”或“地方民俗学生”等角色。
