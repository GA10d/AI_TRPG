# The Silence（深化重构版）/《湖岸录像谜灵》

## 剧本必须项

### 1. 剧本元信息

**剧本名**：The Silence / 《湖岸录像谜灵》  
**剧本 ID / 版本号**：THE_SILENCE_REBUILD_V2  
**题材标签**：美式恐怖、悬疑调查、废墟探索、灵异、录像诅咒、生存压迫、旧案追凶  
**风格标签**：压抑、潮湿、写实、慢性恐惧、声画错位、情感对质、高风险  
**适用人数**：单人  
**角色创建方式**：自由创建；固定前提为“玩家因一份匿名寄来的录像线索，被引至常青湖营地”  
**推荐流程长度**：中篇  
**推荐节奏**：前慢后紧；前段建立气味与空间压迫，中段以线索拼接推进，后段情绪与生存双重升温  
**推荐主持风格**：重氛围、强引导、隐藏判定、允许轻度保护性兜底，但不替玩家做核心判断  
**内容边界提示**：含火灾死亡、窒息、求救未应、失聪、幻听、创伤记忆、道德共犯、背叛、濒死画面、角色牺牲、幸存者负罪

### 2. 故事 Intro（用于显示）

1987 年 7 月 15 日，常青湖营地闭营夜。营地活动中心后方的储物间失火，一名年轻辅导员克莱尔·怀特死于浓烟。官方结论写得很轻：线路短路，逃生不及，纯属意外。营地次年永久关闭，录像厅停摆，码头烂进湖里，像一段再也没人愿意重看的片尾。

但真正被埋掉的，从来不是一场火。

三十多年后，你收到一盘没有寄件人的旧 VHS、一张被烟熏过的拍立得，以及一句潦草的手写留言：

**“她不是死于火。她死于有人听见了，却没有开门。”**

傍晚，你抵达废弃的常青湖营地。入口处，一个叫莉莉·卡特的年轻女孩正拿着半张旧地图等人——她在找失踪的姐姐简；树林里似乎有人正躲在暗处观察这里；而那座本该断电三十多年的“星尘录像厅”，正隔着雾气传来沉闷、规律、像指节拼命砸门一样的响声。

很快你会发现，这个故事真正的怪物不是鬼。

鬼只是结果。  
录像带只是媒介。  
真正的核心，是一条被活人共同改写、共同回避、共同维护了三十多年的责任链：

**是谁先发现了真相？**  
**是谁为了掩盖真相，把“丑闻”改写成“事故”？**  
**是谁点燃了火？**  
**又是谁明明停在门外，听见求救，却选择了后退？**

你要在午夜前进入这片被潮湿森林和旧影像噪点包住的废墟，拼合两盘录像带、一个死者被截断的证词、一名失踪者留下的调查笔记，以及一个幸存者腐烂多年的良知。

最后，你必须决定：

**让真相被倾听，还是让沉默再次赢一次。**

### 3. 玩家角色定位

**玩家身份**：自由创建

**建议身份方向**：

- 调查旧案者
- 收到匿名录像带的人
- 失踪者亲友或受托人
- 记者、纪录片创作者、播客主持人、独立调查员
- 与音频、影像、档案修复有关的从业者
- 误入者
- 对 1987 年事件存在私人动机的人

**身份标签**：外来见证者 / 被引来者 / 可能成为新的记录者  
**社会位置**：默认在营地旧关系网之外，但允许与“简”“旧录像”“地方传闻”存在弱联系  
**初始处境**：黄昏抵达废营地，立刻卷入一场围绕失踪、幽灵、旧录像与火灾责任展开的调查  
**初始认知范围**：

- 知道 1987 年这里发生过火灾
- 知道近期有人重查旧案后失踪
- 知道匿名录像带与营地火灾有关
- 不知道克莱尔、简、塔克、哈罗德之间的完整关系
- 不知道最致命的真相并不是“谁放了火”，而是“谁听见了求救却沉默”

**初始目标**：活着探索营地，找到简，弄清 1987 年火灾与录像带的关系，并决定是否公开真相  
**初始限制**：

- 夜幕持续推进
- 场景危险会升级
- 玩家可能遭遇失聪、耳鸣、空间错位、错误引导
- 无法立刻判断莉莉与塔克谁更值得信任
- 某些线索必须在高压状态下才会显现，但高压也意味着更接近死亡

**初始优势**：

- 玩家背景自由，可用不同方法切入同一真相链
- 可携带自定义道具与专业能力
- 作为外来者，不天然受旧营地“沉默共识”绑架
- 只要愿意倾听，克莱尔会把玩家视为可能的见证人，而不只是猎物

**是否固定主角**：否  
**是否允许自定义背景**：是  
**是否有预设关系网**：默认无；可在创建时与“简”“匿名录像带”“地方档案”建立弱联系  
**与核心事件的关系**：被动卷入的见证者，逐步成长为替死者发声的人  
**与关键 NPC 的初始关系**：

- 与莉莉·卡特：陌生人，迅速形成脆弱同盟
- 与塔克·沃克：未知且危险，对方可能在暗中观察玩家
- 与克莱尔·怀特：未见面，但她已通过录像与异象筛选玩家
- 与简·卡特：未见面，但她其实是匿名线索真正的寄出者

**玩家知道什么 / 不知道什么**：  
玩家知道这里不只是“闹鬼”。  
玩家不知道，这个地方最深的诅咒不是怨灵，而是那一夜所有活着的人共同决定：**先让事情安静下来。**

### 4. 核心目标

**显性目标**：  
探索常青湖营地，查明 1987 年火灾真相，找到简的下落，并设法活着离开。

**隐性目标**：  
修复那一夜被人为切断的“证词链”，让克莱尔的死亡不再被定义成一场轻飘飘的事故；同时避免把她的遭遇再次消费成一段猎奇录像。

**阶段目标**：

1. 进入营地并与莉莉形成初步合作
2. 在辅导员宿舍取得克莱尔的私人痕迹，确认她不是“背景死者”，而是一个具体的人
3. 在旧广播室 / 管理员办公室建立“经费挪用 + 安全失修”线索
4. 进入星尘录像厅，接触黑色录像带与关键值班信息
5. 在湖滨码头取得简留下的调查材料，建立第二条证据链
6. 与塔克接触，判断他是活证人、拖累，还是坏结局入口
7. 找到烧毁储物间，拼出火灾当夜真实顺序
8. 在午夜前进入终局，决定真相要被“播放”、被“承认”，还是再次被“删除”

**成功条件**：

- 玩家拼出一条完整且自洽的责任链
- 克莱尔的死亡得到命名与承认，而非再次被掩埋
- 至少达成一个成立的结局
- 玩家没有被常青湖营地永久吞没
- 真相没有在终局中被彻底篡改成“更好传播但更失真”的版本

**失败条件**：

- 未进入终局即迎来午夜闭锁
- 玩家被拖入湖中或储物间残骸并失去主动性
- 真相链被彻底毁坏，玩家无法再做有效判断
- 玩家让唯一仍可被保存的证词媒介同时毁灭

**不可逆损失条件**：  
以下条件不是“立刻断本”，而是**路线锁死 / 结局降级 / 情感收束变苦**：

- 莉莉死亡
- 黑色录像带完全损毁
- 简的调查资料大面积遗失
- 塔克带走关键证物逃离
- 玩家感官剥夺达到临界后未恢复
- 玩家主动用剪辑、伪造、断章取义方式改写责任链

### 5. 剧本主推进轴

**主推进轴**：调查推进 + 时间压力 + 情感对质 + 生存风险  
**次推进轴**：探索解谜 + 信任判断 + 录像播放机制 + 证词闭环

这部剧真正推动故事向前的，不是“鬼会不会跳出来”，而是五个不断收紧的问题：

- 第一份真相证据是谁留下的
- 谁把真相从“丑闻”改写成了“事故”
- 谁在起火前参与了对克莱尔的围堵与施压
- 谁在拍门声响起时停下了，却没有开门
- 玩家在见证一切之后，究竟会成为发声者，还是下一位沉默者

鬼魂是形式，  
录像带是媒介，  
**沉默才是这部剧真正的怪物。**

### 6. 场景 / 节点结构

#### 节点 1：入口广场

**节点类型**：开场 / 信息枢纽  
**节点功能**：建立 VHS 恐怖气质，引入莉莉与匿名录像带，投放第一层诱饵，交代“有人已经先你一步来过这里”  
**场景描述**：  
生锈的营地木牌半埋在高草里，字迹被雨水洗得发白。电话亭玻璃裂成蛛网状，像一个被掏空的头骨。远处，星尘录像厅的霓虹只剩 “NO SHOW” 两个单词会偶尔抽搐般闪一下。空气里有潮木、铁锈和长年未清的霉水味，像某段被封存太久的地下录音突然被重新播放。  
莉莉站在门口，手里拿着半张旧地图、一张拍立得复印件，以及一盒已经按下“PLAY”键却没有声音的录音笔。她看起来像等了很久，也像不确定玩家会不会真的来。

**可见元素**：

- 生锈营地木牌
- 破碎电话亭
- 公告栏残页
- 星尘录像厅残闪招牌
- 莉莉
- 泥地里的轮胎印与旧脚印
- 一只被雨泡烂的录像带盒

**可交互对象**：

- 电话听筒
- 公告栏上的火灾残页
- 莉莉手中的地图与照片
- 玩家收到的匿名 VHS
- 地上的录像带盒

**可获得信息**：

- 营地于 1988 年永久关闭
- 简失踪前最后出现地点在湖区和录像厅之间
- 电话中残留一段被反复覆盖的录音：“别放那盘带子……别让他再说是短路……”
- 莉莉知道姐姐不是来“探险”，而是来找一件她们母亲一直回避的旧事
- 有人近期把入口附近的杂草清开过，说明除了玩家与莉莉之外，还有第三方在活动

**可触发事件**：

- 电子设备出现雪花屏或录音串音
- 莉莉主动请求同行
- 玩家第一次听见不合理的拍门声
- 树林深处出现疑似监视者的移动影子
- 电话在无人触碰时突然回拨，只有沉重呼吸与三下敲击声

**进入条件**：默认开场  
**离开条件**：决定深入营地  
**节点风险**：低  
**指向下一步的钩子**：

- 地图缺失部分指向宿舍区、旧广播室与码头
- 莉莉提到简曾在语音里说过：“要先找到她的房间，再找他撒过的谎。”

---

#### 节点 2：辅导员宿舍区

**节点类型**：探索 / 第一惊吓 / 人物建构  
**节点功能**：建立克莱尔不是“背景死者”，而是具体而完整的人；投放第一批情感线索；触发第一次感官剥夺  
**场景描述**：  
三间木屋沿泥地排开，像三副被雨水泡涨的棺材。宿舍区几乎没有风，可门帘、挂绳和照片边角却会自己轻轻晃动。  
1 号木屋上锁，窗台压着一张拍立得合影。  
2 号木屋是克莱尔的房间，墙上夏令营海报被人一笔一笔划烂，梳妆台前的裂镜像被谁一拳砸穿。  
3 号木屋有近年使用痕迹，床下有酒瓶和空罐，衣柜里挂着一件旧橄榄球队外套与一条发黄毛巾，像有人反复回到这里，却从没真正敢住整夜。

**可见元素**：

- 裂镜
- 拍立得合影
- 焦黑金属发带碎片
- 一本“声音采集本”
- 旧队服
- 门板背面的抓痕
- 未寄出的信封
- 录音磁带盒

**可交互对象**：

- 裂镜
- 床垫
- 抽屉
- 梳妆台
- 衣柜
- 合影
- 声音采集本
- 旧信件

**可获得信息**：

- 克莱尔并非单纯“孤僻怪人”，她出身本地，父亲曾是旧露天影院放映员，母亲后来逐渐失聪；她从小就习惯用录音和影像留住别人来不及听清的话
- 她在营地负责活动记录和录像放映，原本计划秋天申请纪录片专业
- 简在童年时就认识克莱尔，且并非偶然碰面，而是被她认真倾听过、安慰过、鼓励过
- 克莱尔一直在为某些孩子录“勇敢说话”的练习带，其中一盘上写着字母 “J.”
- 有人这些年反复回来住在 3 号木屋
- 未寄出的信暗示：克莱尔掌握了管理员与营地经费异常的证据

**可触发事件**：

- 触碰裂镜或发带碎片，触发**第一惊吓**
- 玩家短暂失聪，世界进入“静音”状态
- 镜中克莱尔出现，做出“嘘”的手势
- 镜面浮现血字：**“不是火先来的，是门先关上的。”**
- 若翻到“声音采集本”后几页，会听见一段儿童练声录音，其中简童年的声音在杂音里说：“如果我没说出来，你会替我记住吗？”

**进入条件**：进入营地后可达  
**离开条件**：取得发带碎片 / 发现塔克曾在此藏身 / 被惊退  
**节点风险**：中  
**指向下一步的钩子**：

- 旧信与队服都指向塔克
- 合影与莉莉的照片能拼出简和克莱尔的真实旧缘
- 镜中文字指向储物间与录像厅
- 采集本里多次提到“广播室的钥匙”和“他不肯报修的线路”

#### 节点 3：旧广播室与管理员办公室

**节点类型**：探索 / 设定补完 / 中层转折  
**节点功能**：建立“制度性掩盖”这一层真相；让哈罗德不再只是工具人反派，而是责任链第一环；补完营地历史与录像诅咒来源  
**场景描述**：  
入口广场后方是一栋半塌的木制管理楼，旁边连着一间早已停用的广播室。玻璃窗内贴着“保持安静”“紧急时听从指令”的旧标语，字迹因潮湿扩散得像在出汗。广播控制台上落满灰，推子却有一条新近划开的指痕。管理员办公室里堆着捐款名册、维护单据和发霉公函，一整面墙上的钥匙牌只剩下“储物间”“投影间”“广播室”几个钩子是空的。

**可见元素**：

- 广播麦克风
- 老式录音带机
- 营地维护单
- 捐款名册与账本
- 钥匙牌
- 一封没有寄出的举报信
- 儿童应急疏散手册
- 墙上的营地历史照片

**可交互对象**：

- 麦克风
- 录音带机
- 账本
- 文件柜
- 信封
- 钥匙牌
- 疏散手册
- 历史照片背面

**可获得信息**：

- 哈罗德·贝恩不仅是管理员，还是将旧露天影院仓库改造成星尘录像厅的人；他极度看重营地声誉，并长期挪用维护经费填补营地财务窟窿
- 营地多年来明知线路老化、消防器材失效，却始终只做纸面维护
- 一份录音里，哈罗德在闭营夜通过广播要求值班人员“别让孩子听见哭喊，先把场面压住”
- 克莱尔已经写好举报信，收件方是地方报社与县消防办公室，但信并未寄出
- 钥匙记录表显示，储物间外锁与广播室备用钥匙在火灾前后都被人为调换过
- 录像厅之所以会成为“沉默场”的核心，不只是因为死过人，还因为那里原本就是旧放映仓库，隔音墙、磁带机、线路与湖边雾气共同把那一夜未被承认的声音留了下来

**可触发事件**：

- 玩家启动广播台后，整座营地短暂回响哈罗德失真的命令：“安静——所有人都安静——”
- 窗玻璃中短暂出现一个男人的背影，拿着钥匙转身离开
- 若带着发带碎片进入此处，广播会自动播放一段未公开的测试录音：克莱尔正在练习如何把举报内容说得更清楚
- 若莉莉在场并听到举报信内容，她会第一次意识到姐姐并不是“被鬼故事吸引”，而是在追一件真正有人做错过事的旧案

**进入条件**：自由可达，或由宿舍中的采集本指引  
**离开条件**：带走账本复写页 / 发现广播录音 / 被异象惊退  
**节点风险**：中  
**指向下一步的钩子**：

- 维护单与举报信将怀疑指向哈罗德
- 钥匙记录说明“锁门”是现实行为，不是单纯幻象

#### 节点 8：终局节点（码头 / 录像厅）

**节点类型**：收束 / 对峙 / 结局  
**节点功能**：让玩家真正决定“真相是被倾听，还是被再次消费”；完成情感与责任的最后命名  
**场景描述**：  
终局分为两类空间：

**A. 湖滨码头终局**  
风平得不自然，湖面像一整张等待显影的底片。这里更接近安抚、告别与救回。真相在这里不是“曝光”，而是“承认”；不是叫更多人看见，而是终于让该听见的人听见。

**B. 星尘录像厅终局**  
放映机自启，银幕与现实互相污染。这里更接近揭露、逼供与冲突。真相在这里不是“被理解”，而是“被放映”；不是被抚平，而是被撕开。

**可触发事件**：

- 塔克抢带 / 断电 / 毁证
- 克莱尔从银幕、水面或反光中完全显形
- 莉莉牺牲或幸存
- 简的状态被最终揭示
- 玩家必须决定：播放哪一盘、先播放哪一盘、是否要求塔克当面承认、是否优先保人、是否把真相留给私下承认还是带出营地公之于众

**进入条件**：

- 进入标准终局：任一发带部件线完成 + 至少一盘录像带在手 + 玩家对责任链已有有效判断
- 进入完整终局：发带集齐 + 双录像齐备 + 至少取得一条塔克责任物证 + 至少一条哈罗德责任线索
- 若已进入终局区域后才到午夜：不直接断本，而是强制转入高压坏结局 / 苦涩结局判定

**离开条件**：进入结局  
**节点风险**：极高  
**指向下一步的钩子**：直接进入结局结构

### 7. 关键实体

#### 克莱尔·怀特

**类型**：敌对者 / 受害者 / 场景核心幽灵  
**表层印象**：蓝裙、湿发、做“嘘”手势，靠近时会夺走人的听觉与方向感  
**外貌**：blue vintage dress, wet long dark hair, pale skin, burned edges clothing, smoky stains, hollow eyes, dark circles, finger on lips gesture, barefoot, water dripping, VHS glitch effect, analog horror, ghost girl, reflection distortion
**前史补充**：  
她不是凭空长出来的“女鬼素材”。她生前是个很会听的人。父亲托马斯曾是湖边旧露天影院的放映员，母亲艾琳在纺织厂长期受噪音影响，听力一年年衰退。克莱尔从很小就学会如何用磁带、影像和口型去留住会被现实漏掉的声音。她在营地负责活动记录与放映，看起来安静，实则一直在替那些说不利索、来不及说、没人愿意认真听的孩子保存证词。她与童年的简因此结缘。  
**真实动机**：她要的不是单纯复仇，而是让那一夜每一个“听见了却不说”的人被迫面对自己的沉默  
**立场**：对噪音、毁证、逃避真相者极端敌对；对愿意倾听并承认真相的人存在被安抚的可能  
**与玩家关系**：把玩家视作新的见证者，也可能视作新的替代品  
**知情范围**：知道自己如何死；知道哈罗德锁门、塔克后退；不知道简后来把真相追到了哪一步  
**对玩家隐瞒的信息**：她真正最恨的不是“火”，甚至不完全是“锁门”，而是门外那一秒明明有人停下了，最后却还是走了  
**可提供的信息 / 资源**：幻象、镜像文字、音频失真、死亡片段、路线提示、情绪真相  
**会制造的阻碍**：失聪、模仿求救、伪造字条、空间错位、拖拽、对玩家“只取证不倾听”的惩罚  
**行为触发逻辑**：高噪音、毁证、跳过画面、切断证词、接近真相核心  
**变化条件**：发带集齐、双带对照、塔克认罪、简被承认、玩家拒绝把她变成猎奇素材  
**退场条件**：被安抚 / 真相被命名 / 玩家失败并成为新的轮回记录

#### 莉莉·卡特

**类型**：同伴 / 情感支点 / 关系对象  
**表层印象**：年轻、紧绷、明显睡眠不足，努力让自己显得有用
**外貌**：a young american sweetheart girl, blonde hair in a slightly messy high ponytail, soft freckles on nose and cheeks, pale skin, light blue eyes with subtle dark circles, exhausted but determined expression, wearing a light blue oversized hoodie layered under a faded denim jacket, slim dark jeans, dirty white sneakers, holding a polaroid camera hanging around her neck, gripping a flashlight
**前史补充**：  
她是简同母异父的妹妹，比简小很多。她成长时，家里总绕着一个不能提的词转：常青湖。简长大后越来越沉迷旧磁带、旧照片和地方档案，莉莉一度觉得姐姐把死人看得比活人重要。她来营地，不只是为了找人，也是为了证明简没有疯，也证明自己不是永远被排除在家庭秘密之外的小孩。  
**真实动机**：找到失踪的简，同时证明姐姐不是疯了，而是真的接近了某个被掩盖的真相

---

#### 触发器：播放录像带

**触发后事件**：画面主动补全部分真相，但也会召唤或强化克莱尔  
**影响范围**：信息推进、风险升级、结局分歧  
**是否可逆**：不可逆  
**额外变化**：交叉播放双带能得到最完整真相，但风险最高；错误剪辑或中断会明显提高坏结局概率

---

#### 触发器：回应水面中的简

**触发后事件**：简的求救变得具体，可进入救回线  
**影响范围**：码头终局、莉莉关系线、玩家受伤风险  
**持续时间**：中  
**是否可逆**：部分可逆  
**额外变化**：若玩家在证据不足时过早追逐简，可能被沉默场拉偏路线

---

#### 触发器：日落

**触发后事件**：湖区与宿舍异常增强，克莱尔实体化显著上升  
**影响范围**：全局  
**是否可逆**：不可逆  
**额外变化**：所有光源效果下降，广播 / 电话 / 录像失真频率上升

---

#### 触发器：午夜

**触发后事件**：营地闭锁，出口消失  
**影响范围**：全局  
**是否可逆**：不可逆  
**说明**：

- 若此时尚未进入终局，直接进入失败判定
- 若已在终局中，则强制升级为高压坏结局 / 苦涩结局判定

### 10. 风险与后果

**核心风险源**：

- 克莱尔的显灵与感官剥夺
- 塔克的毁证、误导与背刺
- 时间流逝
- 录像播放的召唤效应
- 错误站队
- 在高压场景中过度逗留
- 玩家把真相当“内容素材”而不是“需要承担的证词”

**风险升级方式**：

- 拖延
- 高噪音行动
- 反复播放录像
- 对塔克施压过猛却准备不足
- 长时间忽视莉莉与简的情感线
- 拿到关键证物后迟迟不进入收束
- 自以为掌握真相就开始断章取义地下判断

**即时后果**：

- 失聪、耳鸣、方向混乱
- 被追逐、被拖拽
- 同伴失散
- 关键道具掉落
- 现场受伤
- 录像画面受损或现实污染

**延迟后果**：

- 莉莉不再完全信任玩家
- 塔克提前布局或抢先毁证
- 简的求救窗口缩短
- 光源、录音设备、照片等资源耗尽
- 某些本可通过对话完成的环节，被迫升级为高压冲突

**隐性后果**：

- 玩家越把这一切当成“取证任务”，越容易失去安抚路径
- 玩家若持续替塔克保留借口，终局更易导向毁证与轮回
- 玩家若持续利用莉莉，牺牲结局概率显著提升
- 玩家若只想公开真相、不愿承担真相的完整性，容易触发“删改版真相”结局

**长期后果**：

- 真相链残缺
- 结局降级
- 克莱尔无法真正平息
- 简无法回归
- 玩家成为新的“记录素材”或新的匿名寄件人

**对剧情推进的影响**：  
风险不阻止推进，而是让每一次推进都更贵。玩家总能继续向前，但会因此失去某些人、某些版本的真相，或某种更温和的收束方式。

**对结局的影响**：  
决定玩家能否达成救赎、生还、苦涩、轮回、删改真相或隐藏结局。

### 11. 分支点

#### 分支点 1：先去旧广播室 / 录像厅，还是先去码头

**触发时机**：第一惊吓之后  
**玩家可做选择**：

- 先去旧广播室 / 录像厅
- 先去码头

**直接后果**：

- 先去广播室 / 录像厅：更早接触硬证据和塔克
- 先去码头：更早建立简线和莉莉信任

**延迟后果**：

- 先码头更利于救回 / 安抚
- 先厅线更利于快推真相，但塔克更容易敌化

**影响对象**：莉莉、塔克、终局路线  
**是否可回头**：可  
**是否为隐性分支**：否

---

#### 分支点 2：是否向莉莉完整公开线索

**触发时机**：拿到广播录音、简日记或克莱尔采集本后  
**玩家可做选择**：

- 及时共享
- 选择性共享
- 暂时隐瞒

**直接后果**：

- 共享：莉莉更快成长为真正同伴
- 选择性共享：维持效率，但埋下不信任
- 隐瞒：短期减少情绪拖累，长期关系线受损

**延迟后果**：

- 共享更利于牺牲规避与救回线
- 隐瞒过久会让莉莉在终局质疑玩家是否也在“替别人决定什么该被听见”

**影响对象**：莉莉、码头终局、结局情感重量  
**是否可回头**：部分可  
**是否为隐性分支**：是

---

#### 分支点 3：是否立刻对质塔克

**触发时机**：拿到值班表、烟蒂或求救字条后  
**玩家可做选择**：

- 立刻质问
- 暂时隐瞒
- 诱导合作
- 假装只追哈罗德

**直接后果**：

- 质问：塔克敌意上升
- 隐瞒：可换短期合作
- 诱导：可获得储物间与时间线补充
- 假装只追哈罗德：塔克短暂放松，但终局更可能尝试把责任全推给死人

**延迟后果**：

- 施压过猛会逼出逃跑或毁证
- 隐瞒过久会埋下终局背叛
- 诱导得当可换取更完整供述
- 假装只追哈罗德容易进入“删改真相”路线

**影响对象**：塔克、储物间路线、录像厅终局  
**是否可回头**：部分不可  
**是否为隐性分支**：是

---

#### 分支点 4：优先保人还是优先保证据

**触发时机**：高压追逐、莉莉遇险、简求救出现时  
**玩家可做选择**：

- 优先保莉莉
- 优先抢救证据
- 优先回应简的求救

**直接后果**：

- 保人：证据链可能残缺
- 保证据：关系线受损
- 回应简：可开救回线，但风险高

**延迟后果**：

- 影响谁愿意在终局陪玩家站到最后
- 影响结局情感重量与幸存人数
- 也影响玩家最终如何理解“见证”与“拯救”的边界

**是否可回头**：不可  
**是否为隐性分支**：否

---

#### 分支点 5：终局地点与播放顺序

**触发时机**：至少形成一条有效责任链后  
**玩家可做选择**：

- 码头终局 / 录像厅终局
- 先放简的带子 / 先放黑带 / 两盘交叉播放 / 不播放而直接对质

**直接后果**：

- 简带优先：偏安抚与救回
- 黑带优先：偏揭露与冲突
- 交叉播放：最完整但风险最高
- 不播放：更依赖对话、供述与玩家判断

**延迟后果**：  
影响塔克崩溃方式、克莱尔敌意、简能否回来，以及真相究竟会以“纪念”还是“放映”的方式收束

**是否可回头**：基本不可  
**是否为隐性分支**：否

---

#### 分支点 6：是否要求塔克当面承认

**触发时机**：终局对峙  
**玩家可做选择**：

- 强迫认罪
- 只公开证据
- 放任他逃
- 以“承认部分责任”交换生路
- 把所有责任都推给哈罗德

**直接后果**：

- 认罪可能直接安抚克莱尔，也可能触发暴走
- 只公开证据可达成较弱版本收束
- 放任逃跑更易进入开放后患
- 交换逼供可换生还，但带来道德阴影
- 全推给哈罗德最容易快速收束，但真相将被再次删改

**是否可回头**：不可  
**是否为隐性分支**：是

### 12. 结局结构

#### 结局一：真相救赎

**结局类型**：真相达成 / 生还 / 和解  
**达成条件**：

- 发带集齐
- 双录像带至少完成一次有效对照
- 塔克承认自己听见了求救
- 莉莉存活
- 玩家在终局中明确命名“锁门、起火、沉默”是复合责任
- 玩家没有以断章取义方式篡改责任链

**必要前提**：责任链基本完整  
**禁止条件**：终局外才到午夜；双带全毁  
**收束对象**：克莱尔、塔克、莉莉、简、玩家  
**最终揭示内容**：  
克莱尔死于三重沉默：哈罗德锁门，塔克参与围堵并在火源失控后没有回身开门，其他人则共同把求救改写成事故。  
**主题落点**：真正的驱鬼，是给罪责命名。  
**未完成支线如何收束**：哈罗德更深层的旧账与营地历史可保留余味，但不动摇主真相

---

#### 结局二：湖上归人

**结局类型**：苦涩生还 / 情感达成 / 救回  
**达成条件**：

- 选择码头终局
- 优先回应简的求救
- 克莱尔被安抚
- 塔克未必完整认罪，但真相已被玩家掌握
- 莉莉至少与玩家保持信任

**必要前提**：简线完成度高  
**禁止条件**：莉莉早死；简带损毁  
**收束对象**：简、莉莉、克莱尔、玩家  
**最终揭示内容**：  
简被从“沉默场”中救回，但无法完全恢复成正常现实中的人。她能离开营地，却永远带着湖水般的空白与录音磁噪般的停顿。  
**主题落点**：有人回来了，但没有人能真正回到从前。  
**未完成支线如何收束**：塔克责任可留给后日谈处理，克莱尔得到部分平息

---

#### 结局三：寂静牺牲

**结局类型**：牺牲 / 苦涩真相达成  
**达成条件**：

- 莉莉为救玩家或简牺牲
- 玩家独自完成最终收束
- 真相链仍保留足够完整

**必要前提**：莉莉关系线成立  
**禁止条件**：玩家彻底放弃找人  
**收束对象**：莉莉、克莱尔、塔克、玩家  
**最终揭示内容**：  
最愿意倾听的人失去了声音，真相却终于被听见。  
**主题落点**：真相抵达时，往往已经太晚。  
**未完成支线如何收束**：简可被救回或仅留下影像告别；塔克结局可随终局混乱浮动

---

#### 结局四：删改版真相

**结局类型**：半成功 / 开放 / 道德阴影  
**达成条件**：

- 玩家保住核心证据并活着离开
- 但终局中主动删改、弱化或重组责任链
- 例如：把责任全部推给哈罗德、故意省略塔克的停步沉默，或只留能“更容易传播”的版本

**必要前提**：至少掌握一条完整证据链  
**禁止条件**：玩家完全没有真相  
**收束对象**：玩家、塔克、克莱尔、公众  
**最终揭示内容**：  
社会终于知道常青湖不是“单纯事故”，但它知道的版本仍然不完整。克莱尔暂时安静了，却没有真正离开；她只是再一次被别人决定该如何被讲述。  
**主题落点**：真相一旦被剪辑，也会变成另一种沉默。  
**未完成支线如何收束**：简可能无法接受，莉莉与玩家的关系也可能永久留下裂缝

---

#### 结局五：背叛轮回

**结局类型**：失败 / 恐怖开放 / 轮回  
**达成条件**：

- 过度信任塔克
- 关键证据在终局被毁
- 午夜压力完全压垮局势
- 克莱尔无法被安抚

**必要前提**：真相链残缺  
**收束对象**：玩家、塔克、克莱尔  
**最终揭示内容**：  
录像厅开始播放属于玩家自己的“死亡版本”；下一盘寄往外界的带子里，玩家成了那个不断拍门的人。  
**主题落点**：沉默只会寻找新的容器。  
**未完成支线如何收束**：简永留湖中，莉莉生死不明，塔克可能逃离也可能被拖走

---

#### 结局六：隐藏结局·静音模式

**结局类型**：动态 / 黑暗 / 异化  
**达成条件**（满足其一或组合）：

- 烧毁双带
- 带着发带强行离开
- 与塔克共谋伪造新版本
- 放弃找简
- 玩家主动选择与克莱尔同化
- 玩家死亡但完成了一段“替代录制”

**最终揭示内容**：  
根据玩家行为生成后日谈：  
可能是社会继续相信“事故”；可能是玩家在未来某天收到一盘来自自己的录像；也可能是常青湖从此多出一名新的幽灵记录者。  
**主题落点**：遗忘也是一种谋杀。  
**未完成支线如何收束**：全部以缺席、噪点、未寄出的信和匿名包裹形式收尾

### 13. 主持约束 / Agent 约束

**是否允许强引导**：允许，在玩家明显停滞时通过环境异象、莉莉提示、录音串音、湖面倒影推动  
**是否允许主动补关键线索**：允许，尤其是值班表、发带、双录像带、账本复写页、求救字条  
**是否允许保护性兜底**：允许轻度兜底，避免一次误判直接断本  
**是否允许模糊判定**：允许，且推荐  
**是否允许压缩场景**：允许，但不得跳过以下体验：

- 第一惊吓
- 旧广播室建立哈罗德线
- 录像厅接触黑带
- 码头建立简线
- 塔克对质
- 烧毁储物间幻象
- 终局对峙

**哪些信息不能提前说**：

- 哈罗德是锁门者
- 塔克听见了敲门却没开
- 简尚未彻底死亡
- 克莱尔最深的执念不是杀人，而是让活人承认沉默
- 玩家如果删改真相会触发专门结局

**哪些结论不能替玩家做**：

- 信谁
- 先去哪里
- 是否共享线索给莉莉
- 是否播放录像
- 是否要求塔克认罪
- 是否优先保人
- 是否公开真相
- 是否删改真相版本

**是否需要确保某类体验成立**：  
必须确保：

- 声画错位
- 感官剥夺
- 活人比鬼更难对质
- 真相比怪物现身更恐怖
- 每次成功都伴随更深一层接近创伤核心
- 克莱尔必须先被理解，再决定是否被平息

**是否必须保留悬念**：必须  
**是否必须避免某类走向**：

- 不把克莱尔写成纯杀戮鬼
- 不把塔克写成单薄恶人
- 不把简写成纯麦高芬
- 不把哈罗德写成喧宾夺主的现时反派
- 不把玩家写成只能是媒体人
- 不把“公开真相”自动写成最正确答案；关键在于是否完整、是否负责、是否再次消费死者

### 2. 状态 / 计量模块

**感官剥夺等级**：轻 / 中 / 重 / 临界  
**莉莉信任度**：低 / 中 / 高  
**塔克压迫值**：低 / 中 / 高 / 失控  
**克莱尔实体化进度**：0% - 100%

**简回归可能性**：关闭 / 开启 / 稳定

### 3. 特殊机制模块

**录像播放机制**：  
播放录像不是读档，而是主动召唤；画面越接近真相，现实污染越严重。  
黑带偏向火灾与现场视角；简带偏向锁门与见死不救。交叉播放可补完责任链，但也会让克莱尔与沉默场更接近完全降临。

**感官剥夺机制**：  
以失聪、耳鸣、方向丢失、画面黑白化、口型与声音不同步等表现替代常规数值压力。

**沉默场机制**：  
湖区与录像厅在高压时形成“沉默场”，会吞掉声音、时间感与人的轮廓；简被困于此。它保存的是“未被承认的声音”，不是单纯灵体空间。

**证据闭环机制**：  
单一证据只能支持怀疑，必须通过“物证 + 录像 / 证言 + 幻象 / 文书 + 现场残痕”才能形成有效判断。

**删改真相机制**：  
玩家在终局及后日谈中的讲述方式会反过来改变收束质量。  
“说出真相”不自动等于“说对真相”。

### 5. 敌对系统 / 怪物系统

**敌对者名称**：静语女鬼·克莱尔  
**威胁来源**：怨念依附于录像带、发带与未完成证词；被沉默场放大  
**行为模式**：

- 夺走听觉
- 模仿熟人声音
- 在镜面、水面、银幕中出现
- 拖拽玩家进入高危区域
- 在玩家忽视真相时加强攻击
- 在玩家接近真相时加强幻象
- 对“剪辑、快进、跳过、删改”产生强烈敌意

**弱点**：

- 完整发带
- 被承认的责任链
- 简留下的证词
- 塔克的明确承认
- 玩家拒绝再次沉默
- 玩家拒绝把她变成猎奇传播品

**出现场景**：宿舍、广播室、录像厅、码头、储物间、水面、反光面  
**升级条件**：日落、噪音、毁证、错误站队、午夜、断章取义  
**压迫方式**：先切断感知，再污染判断，最后迫使重演  
**退场方式**：安抚、揭露、轮回吸纳新见证者

### 6. 关系网络

**克莱尔 ↔ 简**：  
童年建立真实友谊；简是少数当年愿意认真听她说话、也被她认真听完的人

**简 ↔ 莉莉**：

**克莱尔 ↔ 塔克**：  
共同值班者；曾有过含混的互相欣赏；最深冲突不是起火，而是“你听见了，却没有回来”

陌生协作到共患难，也可能因利用与隐瞒而分裂

合作、利用、逼供、背叛并存

**玩家 ↔ 克莱尔**：

**玩家 ↔ 简**：  
匿名引路人与被迫接棒的后继见证者

### 7. 误导信息 / 红鲱鱼表

**误导 1：莉莉在撒谎**

- 看似支持：她隐瞒家庭关系与拍立得背面的留言
- 实际真相：她隐瞒的是私人家事与恐惧，不是案件主真相
- 如何纠正：简的日记、拍立得合影、后续情绪对话
- 若玩家信了：莉莉信任下降，救回线受损

**误导 2：塔克只是背锅人**

- 看似支持：真正锁门的是哈罗德
- 实际真相：塔克虽非唯一责任人，但他参与围堵、制造起火条件并在求救前后退
- 如何纠正：双录像带 + 求救字条 + 储物间幻象
- 若玩家信了：更易导向毁证、删改真相与轮回

**误导 3：克莱尔只是要杀人**

- 看似支持：她高攻击性、会拖拽、会诱骗
- 实际真相：她更想逼人承认
- 如何纠正：发带、简线、安抚路径、采集本
- 若玩家信了：剧情会滑向纯逃鬼本

**误导 4：简已经死透了**

- 看似支持：只剩录像、日记和倒影
- 实际真相：她被困在沉默场中
- 如何纠正：湖面异象、码头终局
- 若玩家信了：会放弃救人线

**误导 5：只要公开真相就是好结局**

- 看似支持：旧案就该曝光
- 实际真相：真相若被删改、消费、断章取义，也可能构成对克莱尔的二次伤害
- 如何纠正：简的日记、克莱尔反应、删改真相结局
- 若玩家信了：容易进入“删改版真相”

**误导 6：哈罗德一死，一切都结束了**

- 看似支持：最明确的权力施害者已不在场
- 实际真相：真正需要被面对的，是活下来的人如何继续维持沉默
- 如何纠正：塔克供述、求救字条、终局命名
- 若玩家信了：会把活人责任轻轻放过

### 8. 演出文本

**开场白**  
“常青湖营地的大门没有锁，可你还是觉得自己像在闯入什么不该被打开的东西。风吹过来时，带来的不是树叶摩擦声，而像一盘老旧录像带在缓慢倒转。门口那个女孩抬头看你，像终于等到了一个还愿意听的人。”

**第一惊吓文本**  
“镜子里的人比你先一步转过头。她没有张嘴，可你知道她是在叫你安静。下一秒，世界像被剪断了声音——你听不见莉莉，听不见自己的呼吸，只看见镜面上慢慢渗出一句话：不是火先来的，是门先关上的。”

**广播室触发文本**  
“电流先是在墙里窜了一下，接着整座营地像被一只看不见的手按住。旧喇叭里传出男人失真的声音：‘别让孩子听见。先把场面压住。都安静。’那不是命令，更像一套曾经成功过一次的掩埋方法。”

“你们这些后来的人，总觉得真相是件能带走的东西。可有些真相，留在这儿都烂了，拿出去也只会继续臭。”

“我姐不是来玩通灵游戏的……她回来，是因为她小时候就知道这里有件事不对。她说如果这次还没人把那扇门打开，那以后就再也不会有人记得她是怎么死的了。”

**终局逼供台词样例**  
“你可以说火不是你故意放的。你也可以说锁门的不是你。可你没法再说你没听见。她拍门的时候，你停下过。”

“故事终于被讲出来了。报纸有了标题，节目有了流量，公众有了一个足够好懂的凶手。只有你知道，那天夜里真正被删掉的不是几秒画面，而是一个人停在门外、最终却没有回头的那一秒。”

### 9. 示例流程

**标准调查回合示例**：  
入口广场接触莉莉 → 宿舍获得发带碎片并触发失聪 → 玩家决定先去广播室 / 录像厅还是先去码头 → 第一条调查线形成

**关系推进示例**：  
码头找到简日记 → 莉莉情绪崩溃 → 玩家选择安慰、追问或隐瞒 → 决定她终局是否会继续跟随、是否愿意为玩家冒险

**节点切换示例**：  
广播室拿到账本复写页 → 录像厅确认值班表 → 塔克现身 → 玩家选择合作或施压 → 转入储物间真相线

**提示递进示例**：  
第一次只知道“她被锁住了” → 第二次知道“塔克在场” → 第三次知道“哈罗德锁门” → 第四次知道“塔克停下了却没救” → 第五次知道“公开并不等于完整”

**结局收束示例**：  
若双带齐备、发带完整、塔克认罪且玩家不删改责任链，则进入真相救赎；若毁证、拖延、过度信任塔克，则进入背叛轮回；若玩家主动保留能传播但失真的版本，则进入删改版真相

### 10. 变体与扩展

**替换真相**：  
可把哈罗德的动机替换为更深层的地方利益链或教会 / 捐助体系，但首版建议保留“经费挪用 + 安全失修 + 锁门灭证”，逻辑更集中。

**可替换角色**：  
塔克可改写为其他“当年在场的沉默者” archetype，但必须保留其“既有罪又可悔、最致命的一刻是后退而不是点火”的复杂性。

**可替换结局**：  
可增加“真相公开后玩家被媒体吞噬”的后续型结局，进一步强调“被看见”未必等于“被理解”。

**难度档位**：

- 低难：提示更多，录像播放风险更低，莉莉更稳定
- 中难：标准版
- 高难：感官剥夺更强、莉莉更脆弱、塔克更狡猾、午夜更快逼近

**多周目变化**：

- 第二轮可提前让玩家意识到 “NO SHOW” 不只是招牌残字，而是当年的闭营放映主题
- 某些照片会在后续周目显出哈罗德站在远处的身影
- 广播录音会从“安静”逐渐补完为更完整命令
- 隐藏结局可扩展为“玩家成为下一位匿名寄件人”

**单人 / 多人适配差异**：  
当前为单人版；若改多人，可让莉莉或塔克的一部分功能转移给玩家角色之间，例如由一名玩家承担“简旧识”位置，另一名玩家承担“本地人 / 旧案关联者”位置。
