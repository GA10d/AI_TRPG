前端 UI 多语言功能已经接入并完成第一轮收口。

目前界面已经提供 `简体中文 / English / 日本語` 三套 UI 语言包，主菜单右上角新增了语言下拉菜单，玩家可以直接切换界面显示语言。

这次不只是补了一个语言切换入口，而是把前端主要可见文案统一收进语言包，覆盖了：
主菜单、继续游戏、记录、退出、游戏主界面、剧本选择、游戏设置、设置页、分支回溯图，以及 App 内建局提示、占位提示、快速测试等状态文案。

同时修复了旧版日语语言包损坏的问题，重新建立了 `ja_jp` 文件，确保切换到日语后，设置页和主流程页面能跟着一起切换，不再出现“菜单切了语言、页面还停在中文”的情况。

另外，未实现语言包的未来语言代码现在会统一回退到英语，而不会错误回退到中文。
例如后续如果接入新的 UI 语言，但对应语言文件还没有完成，界面会默认显示 `en_us`。

这一轮收口的范围是“前端静态 UI 文案”。
后端接口直接返回的错误信息，以及模型实时生成的剧情文本，不属于 UI 语言包自动翻译范围，后续如果需要，也可以继续补一层错误码与语言包映射。
