# AI TRPG 3.0

Language / 语言：

- [中文 README](./README.zh-CN.md)
- [English README](./README.en.md)

Text-first AI TRPG prototype with a React web client, Node session service, content packs, save/load, replay, and multi-model support.

以文字叙事为核心的 AI TRPG 原型，包含 React 前端、Node 会话服务、内容包、存档回放与多模型接入。
