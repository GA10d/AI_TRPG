# 规则范式  
****必须项****  
1. 世界观  
    *  题材：奇幻、科幻、校园、恐怖、武侠、末日  
    * 玩法重心：战斗、调查、社交、经营、生存、叙事  
    * 风格基调：轻松搞笑、严肃沉浸、高致死、英雄史诗  
    * GM 权限范围：主持人裁定权有多大，是否鼓励即兴判定  
    * 文字式简介  
    * 不一定要把世界观写死，但至少要说明规则和设定如何连接  
        * 魔法/科技在世界中的地位  
        * 普通人和超凡者差距  
        * 阵营、组织、社会秩序  
        * 常见敌人类型  
        * 设定词条如何影响判定  
        * 如果是通用系统，要写“如何把规则套进不同世界”。  
2. 判定机制  
    * 用什么判定：d20、d100、骰池、扑克牌、资源点、无骰叙事  
    * 成功标准：大于等于、低于等于、比对难度、对抗判定  
    * 成功层级：失败、成功、困难成功、大成功、大失败  
    * 优势/劣势机制：重掷、加骰、减骰、取高取低  
3. 伤害、风险与后果  
    * 失败是否仅仅“没做到”  
    * 是否有代价成功  
    * 是否会受伤、失去物品、暴露位置、推进危险  
    * 重伤、创伤、疯狂、污染、腐化等长期后果  
    * 死亡是否常见，是否可逆  
    * 好的规则一般不只是“成功/失败”，还会设计失败推动剧情。  
4. 角色创建  
    * 属性：力量、敏捷、智力、魅力之类  
    * 技能：潜行、说服、射击、医疗、调查等  
    * 职业/种族/背景/模板  
    * 初始资源：生命、装备、金钱、法术、专长  
    * 角色特色：特性、天赋、缺陷、信念、动机  
    * 数值分配方式：点购、掷骰、模板选取、自由构筑  
5. 行动与场景规则  
    * 一般行动：观察、移动、交互、使用物品  
    * 社交场景：谈判、欺骗、威吓、交涉  
    * 探索场景：搜索、追踪、解谜、旅行、生存  
    * 潜行与侦查：是否有视野、警觉、隐蔽机制  
    * 时间推进：回合制、分钟制、场景制、自由叙事制  
****可选项****  
6.  资源系统  
    * 生命值 / 精神值 / 体力值  
    * 法力 / 技能点 / 命运点 / 灵感点  
    * 金钱、补给、弹药、耐久  
    * 社会资源：人脉、声望、信用  
    * 特殊叙事资源：剧情点、命运扭转、编辑权  
7. 成长与奖励机制  
    * 经验值  
    * 里程碑升级  
    * 技能成长  
    * 获得装备、法术、称号、关系网  
    * 心理变化或信念成长  
    * 阵营、声望、派系关系变化  
8. 世界与设定接口  
    * 魔法/科技在世界中的地位  
    * 普通人和超凡者差距  
    * 阵营、组织、社会秩序  
    * 常见敌人类型  
    * 设定词条如何影响判定  
9. 流程示例  
    * 一次完整判定示例  
    * 一轮战斗示例  
    * 一次社交交涉示例  
    * 一次调查推进示例  
    * 角色创建示例
